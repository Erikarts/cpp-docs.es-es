# <a name="ml-nonfatal-error-a2119"></a>Error recuperable A2119 de ML

**se debe especificar el tipo de idioma**

No se proporcionó un tipo de lenguaje a una definición de procedimiento o prototipo.

Se debe declarar un tipo de lenguaje en cada definición de procedimiento o prototipo si no se especifica un tipo de idioma predeterminado. Un tipo de idioma predeterminado se establece mediante [. ](../../assembler/masm/dot-model.md)Directive de modelo, **Option lang**o las opciones de línea de comandos de ml **/GC** o **/GD**.
