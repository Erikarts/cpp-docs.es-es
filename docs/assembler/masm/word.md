# <a name="word"></a>WORD

Asigna e inicializa, opcionalmente, una palabra (2 bytes) de almacenamiento para cada `initializer`. También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.

## <a name="syntax"></a>Sintaxis

> [[*nombre*]] WORD *inicializador* [[, *inicializador*]]...
