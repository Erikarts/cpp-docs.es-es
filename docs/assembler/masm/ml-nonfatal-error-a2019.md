# <a name="ml-nonfatal-error-a2019"></a>Error recuperable A2019 de ML

**el operando debe ser de tipo de registro o campo**

El operando que sigue al operador de [ancho](../../assembler/masm/operator-width.md) o de [máscara](../../assembler/masm/operator-mask.md) no era válido.

El operador [width](../../assembler/masm/operator-width.md) toma un operando que es el nombre de un campo o un registro. El operador [Mask](../../assembler/masm/operator-mask.md) toma un operando que es el nombre de un campo o de un tipo de registro.
