# <a name="ml-nonfatal-error-a2019"></a>Error recuperable A2019 de ML

**operando debe ser de tipo de registro o campo**

La siguiente operando la [ancho](../../assembler/masm/operator-width.md) o [máscara](../../assembler/masm/operator-mask.md) operador no era válido.

El [ancho](../../assembler/masm/operator-width.md) operador toma un operando que es el nombre de un campo o un registro. El [máscara](../../assembler/masm/operator-mask.md) operador toma un operando que es el nombre de un campo o un tipo de registro.
