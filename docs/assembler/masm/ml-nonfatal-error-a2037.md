# <a name="ml-nonfatal-error-a2037"></a>Error recuperable A2037 de ML

**instrucción no permitida dentro de la definición de la estructura**

Una definición de estructura contenía una instrucción no válida.

Una estructura no puede contener instrucciones, etiquetas, procedimientos, directivas de flujo de control, [. STARTUP](../../assembler/masm/dot-startup.md), o [. SALGA](../../assembler/masm/dot-exit.md)de.
