# <a name="perioderridn-perioderridni"></a>&period;ERRIDN, &period;ERRIDNI

Genera un error si los elementos de texto son idénticos.

## <a name="syntax"></a>Sintaxis

> **.ERRIDN**\[**I**] *textitem1* __,__ *textitem2* \[ __,__ *message*]

## <a name="remarks"></a>Comentarios

Si  **&period;ERRIDNI** es dada, la comparación distingue entre mayúsculas y minúsculas.
