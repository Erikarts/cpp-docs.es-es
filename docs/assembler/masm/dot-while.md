# <a name="while-32-bit-masm"></a>. WHILE (32 bits MASM)

Genera código que ejecuta el bloque de *instrucciones* mientras la *condición* sigue siendo true. (solo para MASM de 32 bits).

## <a name="syntax"></a>Sintaxis

> **.**  *Condición* while\
> *instrucciones*\
> **.ENDW**
