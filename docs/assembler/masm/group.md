# <a name="group"></a>GROUP

(solo para MASM de 32 bits). Agregue los *segmentos* especificados al grupo denominado *Name*.

## <a name="syntax"></a>Sintaxis

> *nombre* del *segmento* de **Grupo** ⟦ __,__ *segmento* ... ⟧
