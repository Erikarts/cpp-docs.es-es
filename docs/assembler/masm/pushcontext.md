# <a name="pushcontext"></a>PUSHCONTEXT

Guarda la parte o todo el *contexto*actual: el registro de segmentos presupone el valor de la base, las marcas de lista y CREF, o los valores del procesador/coprocesador. El *contexto* **se puede suponer (** solo para MASM de 32 bits), **base**, **lista**, **CPU** (solo para MASM de 32 bits) o **todos**.

## <a name="syntax"></a>Sintaxis

> *Contexto* de PUSHCONTEXT
