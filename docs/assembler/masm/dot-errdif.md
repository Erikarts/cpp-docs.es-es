# <a name="perioderrdif-perioderrdifi"></a>&period;ERRDIF, &period;ERRDIFI

Genera un error si los elementos de texto son diferentes.

## <a name="syntax"></a>Sintaxis

> **.ERRDIF**\[**I**] *textitem1* __,__ *textitem2* \[ __,__ *message*]

## <a name="remarks"></a>Comentarios

Si  **&period;ERRDIFI** es dada, la comparación distingue entre mayúsculas y minúsculas.
