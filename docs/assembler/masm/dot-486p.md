# <a name="486p-32-bit-masm"></a>.486P (MASM de 32 bits)

Habilita el ensamblado de todas las instrucciones (incluido el privilegio) para el procesador 80486. (solo para MASM de 32 bits).

## <a name="syntax"></a>Sintaxis

> **.486P**
