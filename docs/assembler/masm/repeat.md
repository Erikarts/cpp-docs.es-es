# <a name="repeat"></a>REPETIR

Marca un bloque que se va a repetir en momentos de *expresión* . Igual que [REPT](../../assembler/masm/rept.md).

## <a name="syntax"></a>Sintaxis

> *Expresión* de repetición\
> *instrucciones*\
> **ENDM**
