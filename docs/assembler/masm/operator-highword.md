# <a name="operator-highword"></a>Operador HIGHWORD

Devuelve los 16 bits superiores de los bits 32 inferiores de la *expresión*. Las expresiones MASM son valores de 64 bits.

## <a name="syntax"></a>Sintaxis

> *Expresión* HIGHWORD
