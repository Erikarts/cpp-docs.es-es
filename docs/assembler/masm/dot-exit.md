# <a name="exit-32-bit-masm"></a>. SALIR (MASM de 32 bits)

Genera código de terminación. (solo para MASM de 32 bits).

## <a name="syntax"></a>Sintaxis

> . EXIT [[expresión]]

## <a name="remarks"></a>Comentarios

Devuelve una *expresión* opcional a la shell.
