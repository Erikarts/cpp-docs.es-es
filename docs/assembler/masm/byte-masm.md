# <a name="byte-masm"></a>BYTE (MASM)

Asigna e inicializa opcionalmente un byte de almacenamiento para cada `initializer`.

## <a name="syntax"></a>Sintaxis

> [[nombre]] BYTEinitializer [[, inicializador]]...

## <a name="remarks"></a>Comentarios

También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.
