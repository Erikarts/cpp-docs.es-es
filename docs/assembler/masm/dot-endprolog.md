# <a name="endprolog"></a>.ENDPROLOG

Señala el final de las declaraciones de prólogo.

## <a name="syntax"></a>Sintaxis

> **.ENDPROLOG**

## <a name="remarks"></a>Notas

Es un error usar cualquiera de las declaraciones de prólogo fuera de la región entre el **marco** de [proceso](proc.md) y **. ENDPROLOG**.

Para obtener más información, consulte [MASM para x64 (ml64. exe)](masm-for-x64-ml64-exe.md).
