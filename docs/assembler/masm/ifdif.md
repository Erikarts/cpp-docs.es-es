# <a name="ifdif-ifdifi"></a>IFDIF, IFDIFI

Ensamblado de concesiones si los elementos de texto son diferentes.

## <a name="syntax"></a>Sintaxis

> **IFDIF**\[**I**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Comentarios

Si **IFDIFI** es dada, la comparación distingue entre mayúsculas y minúsculas. Consulte [IF](../../assembler/masm/if-masm.md) para conocer la sintaxis completa.
