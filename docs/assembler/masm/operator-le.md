# <a name="operator-le"></a>Operador LE

Devuelve true (-1) si *expression1* es menor o igual que *expression2*, o devuelve false (0) si no lo es.

## <a name="syntax"></a>Sintaxis

> *expresión1* **le** *expresión2*
