# <a name="tbyte"></a>TBYTE

Asigna y opcionalmente inicializa 10 bytes de almacenamiento para cada *inicializador*. También se puede usar como especificador de tipo en cualquier lugar en el que un tipo sea válido. **DT** es un sinónimo de de un **terabyte**.

## <a name="syntax"></a>Sintaxis

> ⟦*nombre*⟧ de *inicializador* ⟦ __,__ *inicializador* ... ⟧
