# <a name="echo"></a>ECHO

Muestra el *mensaje* al dispositivo de salida estándar (de forma predeterminada, la pantalla).

## <a name="syntax"></a>Sintaxis

> *Mensaje* de eco

## <a name="remarks"></a>Notas

Igual que [% out](percent-out.md).
