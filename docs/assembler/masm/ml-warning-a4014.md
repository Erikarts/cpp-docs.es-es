# <a name="ml-warning-a4014"></a>Advertencia A4014 de ML

instrucciones y datos inicializados no admitidos en segmentos de BSS

Se intentó definir datos inicializados en una sección de BSS.  Una sección de BSS se define como una clase cuyo nombre es BSS.  Esto incluye el `.data?`de segmentos simplificado.
