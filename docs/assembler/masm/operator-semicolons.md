# <a name="operator-"></a>Operador ;;

Trata el *texto* como comentario en una macro que solo aparece en la definición de macro. La lista no muestra el *texto* donde se expande la macro.

## <a name="syntax"></a>Sintaxis

> **;;** *texto* de
