# <a name="operator-"></a>Operador :

Invalida el segmento predeterminado de *expresión* con *segmento*. El *segmento* puede ser un registro de segmento, nombre del grupo, nombre de segmento o expresión de segmento. El *expresión* debe ser una constante.

## <a name="syntax"></a>Sintaxis

> *segmento* : *expresión*
