# <a name="listif"></a>.LISTIF

Inicia la lista de instrucciones en los bloques condicionales falsas.

## <a name="syntax"></a>Sintaxis

> .LISTIF

## <a name="remarks"></a>Comentarios

Igual que [. LFCOND](../../assembler/masm/dot-lfcond.md).
