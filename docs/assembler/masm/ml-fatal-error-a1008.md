# <a name="ml-fatal-error-a1008"></a>Error irrecuperable A1008 de ML

**anidamiento de macro no coincidentes**

Ya sea una macro no se terminó antes del final del archivo o la directiva de terminación [ENDM](../../assembler/masm/endm.md) se encuentra fuera de un bloque de macro.

Una causa de este error es la omisión del punto antes de [. Repita](../../assembler/masm/dot-repeat.md) o [. MIENTRAS](../../assembler/masm/dot-while.md).
