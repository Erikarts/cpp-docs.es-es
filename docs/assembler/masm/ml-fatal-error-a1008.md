# <a name="ml-fatal-error-a1008"></a>Error irrecuperable A1008 de ML

**anidación de macros no coincidentes**

Una macro no finalizó antes del final del archivo o la directiva [ENDM](endm.md) de terminación se encontró fuera de un bloque de macros.

Una causa de este error es la omisión del punto anterior [. Repita](dot-repeat.md) o [. WHILE](dot-while.md).
