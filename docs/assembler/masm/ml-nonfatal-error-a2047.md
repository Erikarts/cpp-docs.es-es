# <a name="ml-nonfatal-error-a2047"></a>Error recuperable A2047 de ML

**cadena vacía (NULL)**

Cadena formada por un par de delimitadores de Comillas y ningún carácter en.

Para que una cadena sea válida, debe contener 1-255 caracteres.
