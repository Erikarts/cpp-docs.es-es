# <a name="686p-32-bit-masm"></a>.686P (MASM de 32 bits)

Habilita el ensamblado de todas las instrucciones (incluido el privilegio) para el procesador Pentium Pro. (solo para MASM de 32 bits).

## <a name="syntax"></a>Sintaxis

> **.686P**
