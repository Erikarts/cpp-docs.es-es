# <a name="ml-nonfatal-error-a2004"></a>Error recuperable A2004 de ML

**conflicto de tipos de símbolos: identificador**

La directiva [EXTERNDEF](externdef.md) o [Label](label-masm.md) se usó en una variable, símbolo, estructura de datos o etiqueta que se definió en el mismo módulo pero con un tipo diferente.
