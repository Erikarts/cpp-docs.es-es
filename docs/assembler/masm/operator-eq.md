# <a name="operator-eq"></a>Operador EQ

Devuelve true (-1) si *expression1* es igual a *expression2*, o devuelve false (0) si no lo es.

## <a name="syntax"></a>Sintaxis

> *expresión1* **EQ** *expresión2*
