# <a name="oword"></a>OWORD

Se usa como especificador de tipo cuando se requiere un tipo de datos de 16 bytes.

## <a name="syntax"></a>Sintaxis

> ⟦*nombre*⟧ **OWORD**  *inicializador* ⟦ __,__ *inicializador* ... ⟧
