# <a name="ml-nonfatal-error-a2189"></a>Error recuperable A2189 de ML

**combinación no válida con alineación de segmento**

La alineación especificada por la directiva **align** o `EVEN` era mayor que la alineación del segmento actual según lo especificado por la directiva **Segment** .
