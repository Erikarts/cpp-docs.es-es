# <a name="operator--masm-run-time"></a>operador ! (Tiempo de ejecución de MASM)

Negación lógica. Se usa solo dentro de [. Si](dot-if.md)es, [. WHILE](dot-while.md), o [. REPETIR](dot-repeat.md) bloques y evaluarlos en tiempo de ejecución, no en tiempo de ensamblado.

## <a name="syntax"></a>Sintaxis

> **!** *expression*
