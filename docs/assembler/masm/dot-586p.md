# <a name="586p-32-bit-masm"></a>.586P (MASM de 32 bits)

Habilita el ensamblado de todas las instrucciones (incluido el privilegio) para el procesador Pentium. (solo para MASM de 32 bits).

## <a name="syntax"></a>Sintaxis

> **.586P**
