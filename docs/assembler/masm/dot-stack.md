# <a name="stack-32-bit-masm"></a>. PILA (MASM de 32 bits)

Cuando se utiliza con [. MODELO](../../assembler/masm/dot-model.md), define un segmento de pila (con la pila de nombres de segmento). El `size` opcional especifica el número de bytes de la pila (el valor predeterminado es 1.024). La Directiva `.STACK` cierra automáticamente la instrucción Stack. (solo para MASM de 32 bits).

## <a name="syntax"></a>Sintaxis

> . PILA [[tamaño]]
