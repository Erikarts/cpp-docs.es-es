# <a name="real8"></a>REAL8

Asigna e inicializa, opcionalmente, un número de punto flotante de precisión doble (8 bytes) para cada *inicializador*.

## <a name="syntax"></a>Sintaxis

> inicializador de REAL8 nombre [[, inicializador]]...
