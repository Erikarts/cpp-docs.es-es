# <a name="errdef"></a>.ERRDEF

Genera un error si *el nombre* es una etiqueta, una variable o un símbolo definidos previamente.

## <a name="syntax"></a>Sintaxis

> **. ERRDEF** *nombre* ⟦ __,__ *mensaje*⟧
