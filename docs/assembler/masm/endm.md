# <a name="endm"></a>ENDM

Termina un bloque de macro o de la repetición.

## <a name="syntax"></a>Sintaxis

> ENDM

Consulte [MACRO](../../assembler/masm/macro.md), [para](../../assembler/masm/for-masm.md), [FORZADA](../../assembler/masm/forc.md), [repita](../../assembler/masm/repeat.md), o **mientras**.
