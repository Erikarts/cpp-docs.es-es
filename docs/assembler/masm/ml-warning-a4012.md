# <a name="ml-warning-a4012"></a>Advertencia A4012 de ML

**información de número de línea de segmento sin que la clase 'CODE'**

Se produjeron las instrucciones de un segmento que no tenía un nombre de clase que termina con "Código". El ensamblador no ha generado la información de CodeView para estas instrucciones.

CodeView no puede procesar los módulos de código en segmentos con los nombres de clase que no terminan con "Código".
