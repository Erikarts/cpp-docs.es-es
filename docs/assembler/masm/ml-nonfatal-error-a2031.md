# <a name="ml-nonfatal-error-a2031"></a>Error recuperable A2031 de ML

**debe ser un índice o un registro base**

Se intentó usar un registro que no era un registro base o de índice en una expresión de memoria.

Por ejemplo, las expresiones siguientes provocan este error:

```asm
[ax]
[bl]
```
