# <a name="ifidn-ifidni"></a>IFIDN, IFIDNI

Ensamblado de concesiones si los elementos de texto son idénticos.

## <a name="syntax"></a>Sintaxis

> **IFIDN**\[**me**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Comentarios

Si **IFIDNI** es dada, la comparación distingue entre mayúsculas y minúsculas. Consulte [IF](../../assembler/masm/if-masm.md) para conocer la sintaxis completa.
