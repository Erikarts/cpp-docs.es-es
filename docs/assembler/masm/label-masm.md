# <a name="label"></a>LABEL

Crea una nueva etiqueta asignando el valor actual de Location-Counter y el *qualifiedType* determinado a *Name*.

## <a name="syntax"></a>Sintaxis

> **etiqueta** de nombre *qualifiedType*
>
> **etiqueta** de nombre ⟦**Near** | **Far** | **proc**⟧ **ptr** ⟦*qualifiedType*⟧
