# <a name="listmacroall"></a>.LISTMACROALL

Inicia la lista de todas las instrucciones de las macros.

## <a name="syntax"></a>Sintaxis

> **.LISTMACROALL**

## <a name="remarks"></a>Notas

Igual que [. LALL](dot-lall.md).
