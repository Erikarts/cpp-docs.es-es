# <a name="operator-ge"></a>Operador GE

Devuelve true (-1) si *expression1* es mayor o igual que *expression2*, o devuelve false (0) si no lo es.

## <a name="syntax"></a>Sintaxis

> *expression1* **GE** *expresión2*
