# <a name="operator-overflow"></a>Operador OVERFLOW?

Estado de la marca de desbordamiento. Se usa solo dentro de [. Si](dot-if.md)es, [. WHILE](dot-while.md), o [. REPETIR](dot-repeat.md) bloques y evaluarlos en tiempo de ejecución, no en tiempo de ensamblado.

## <a name="syntax"></a>Sintaxis

> **OVERFLOW?**
