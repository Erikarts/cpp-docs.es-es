# <a name="public-masm"></a>PUBLIC (MASM)

Hace que cada variable, etiqueta o símbolo absoluto especificado como *nombre* disponibles para todos los otros módulos en el programa.

## <a name="syntax"></a>Sintaxis

> Nombre público [[langtype]] [[, nombre [[langtype]]]]...
