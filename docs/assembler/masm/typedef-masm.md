# <a name="typedef"></a>TYPEDEF

Define un nuevo tipo o **proto** denominado *Name*, que es equivalente a *Type* o *protoDefinition*.

## <a name="syntax"></a>Sintaxis

> *nombre* **TYPEDEF** *Type* | **proto** *protoDefinition*
