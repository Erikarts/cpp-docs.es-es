# <a name="const-32-bit-masm"></a>. CONST (MASM de 32 bits)

Cuando se utiliza con [. MODELO](dot-model.md)inicia un segmento de datos constante (con el nombre de segmento **const**).

## <a name="syntax"></a>Sintaxis

> **. CONSTANTE**\
> ⟦ *segmentItem* ⟧...

## <a name="remarks"></a>Observaciones

Este segmento tiene el atributo de solo lectura.
