# <a name="elseif"></a>ELSEIF

Combina [else](else-masm.md) y [si](if-masm.md) en una instrucción.

## <a name="syntax"></a>Sintaxis

> \ **ELSEIF** *constantExpression*
> *instrucciones*\
> ⟦**ELSE**\
> *else-statements*⟧ \
> **ENDIF**
