# <a name="while-masm"></a>WHILE (MASM)

Repite el ensamblado de *instrucciones* de bloque siempre y cuando la *expresión* siga siendo true.

## <a name="syntax"></a>Sintaxis

> *Expresión* while\
> *instrucciones*\
> **ENDM**
