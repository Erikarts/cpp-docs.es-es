# <a name="while"></a>WHILE

Repite el ensamblado de *instrucciones* de bloque, siempre y cuando *constantExpression* siga siendo true.

## <a name="syntax"></a>Sintaxis

> **While** *constantExpression*\
> *instrucciones*\
> **ENDM**
