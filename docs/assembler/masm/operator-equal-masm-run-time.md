# <a name="operator--masm-run-time"></a>Operador == (En tiempo de ejecución de MASM)

Es igual a. Se usa solo dentro de [. Si](dot-if.md)es, [. WHILE](dot-while.md), o [. REPETIR](dot-repeat.md) bloques y evaluarlos en tiempo de ejecución, no en tiempo de ensamblado.

## <a name="syntax"></a>Sintaxis

> *expression1* **==** *expresión2*
