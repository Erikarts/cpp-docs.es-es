# <a name="operator-lowword"></a>Operador LOWWORD

Devuelve los 16 bits inferiores de la *expresión*. Las expresiones MASM son valores de 64 bits.

## <a name="syntax"></a>Sintaxis

> *Expresión* LOWWORD
