# <a name="ml-nonfatal-error-a2038"></a>Error recuperable A2038 de ML

**falta un operando para el operador de macro**

El ensamblador encontró el final de la lista de parámetros de una macro inmediatamente después del [.](operator-logical-not-masm-run-time.md) o [%](operator-percent.md) operador.
