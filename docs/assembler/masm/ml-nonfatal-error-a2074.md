# <a name="ml-nonfatal-error-a2074"></a>Error recuperable A2074 de ML

**no se puede tener acceso a la etiqueta mediante registros de segmento**

Se intentó obtener acceso a una etiqueta a través de un registro de segmento que no se supuso a su segmento o grupo.
