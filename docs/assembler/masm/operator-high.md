# <a name="operator-high"></a>Operador HIGH

Devuelve los 8 bits superiores de los 16 bits inferiores de la *expresión*. Las expresiones MASM son valores de 64 bits.

## <a name="syntax"></a>Sintaxis

> *Expresión* alta
