# <a name="codesize"></a>Codificar \@

0 para los modelos **minúsculo**, **pequeño**, **compacto**y **plano** , y 1 para **los modelos** **mediano**, **grande y grande**(numérico igual).

## <a name="syntax"></a>Sintaxis

> **\@CodeSize**
