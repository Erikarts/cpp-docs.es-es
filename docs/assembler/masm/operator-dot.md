# <a name="operator-"></a>Operador .

El primer operador devuelve *expresión* más el desplazamiento de *campo* dentro de su estructura o unión. El segundo operador devuelve el valor de la ubicación señalada por *registrar* más el desplazamiento de *campo* dentro de su estructura o unión.

## <a name="syntax"></a>Sintaxis

> *expresión* __.__ *field* \[__.__ *campo*]...
>
> __\[__*registrar*__]__ __.__ *field* \[__.__ *campo*]...
