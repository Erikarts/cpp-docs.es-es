# <a name="ml-nonfatal-error-a2078"></a>Error recuperable A2078 de ML

**la instrucción no permite el direccionamiento indirecto lejano**

Un salto o bucle condicional no puede tomar un operando de memoria. Se debe asignar una dirección o etiqueta relativa.
