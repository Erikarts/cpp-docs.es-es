# <a name="data"></a>.DATA?

(solo para MASM de 32 bits). Cuando se utiliza con [. MODELO](dot-model.md), inicia un segmento de datos Near para datos no inicializados (nombre de segmento _BSS).

## <a name="syntax"></a>Sintaxis

> **. ¿DATOS?** \
> ⟦ *segmentItem* ⟧...
