# <a name="ends"></a>ENDS

Marca el final del segmento, la estructura o el *nombre* de Unión previamente iniciado con el [segmento](segment.md), el [struct](struct-masm.md), la [Unión](union.md)o una directiva de segmento simplificado.

## <a name="syntax"></a>Sintaxis

> *nombre* **finaliza**
