# <a name="datasize"></a>\@Datasize

0 para los modelos **minúsculo**, **pequeño**, **mediano**y **plano** , 1 para los modelos **compacto** y **grande** y 2 para **el modelo de gran tamaño** (numérico igual).

## <a name="syntax"></a>Sintaxis

> **\@Datasize**
