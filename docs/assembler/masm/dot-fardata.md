# <a name="fardata-32-bit-masm"></a>. FARDATA (MASM de 32 bits)

Cuando se utiliza con [. MODELO](dot-model.md), inicia un segmento de datos lejano para los datos inicializados (nombre FAR_DATA o *nombre*del segmento). (solo para MASM de 32 bits).

## <a name="syntax"></a>Sintaxis

> **. FARDATA** ⟦*nombre*⟧
