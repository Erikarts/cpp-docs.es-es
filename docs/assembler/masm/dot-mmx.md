# <a name="mmx-32-bit-masm"></a>. MMX (32 bits MASM)

Habilita el ensamblado de las instrucciones MMX o de una sola instrucción, múltiples datos (SIMD). (solo para MASM de 32 bits).

## <a name="syntax"></a>Sintaxis

> .MMX
