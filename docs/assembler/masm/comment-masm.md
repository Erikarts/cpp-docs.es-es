# <a name="comment"></a>COMMENT

Trata todo el *texto* entre o en la misma línea que los delimitadores como comentario.

## <a name="syntax"></a>Sintaxis

> **Comentario** *delimitador* ⟦*Text*⟧ \
> ⟦*Text*⟧ \
> ⟦*Text*⟧ *Delimiter* ⟦*Text*⟧
