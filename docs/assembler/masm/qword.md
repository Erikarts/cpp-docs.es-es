# <a name="qword"></a>QWORD

Asigna y, opcionalmente, inicializa 8 bytes de almacenamiento para cada *inicializador*. También se puede usar como especificador de tipo en cualquier lugar en el que un tipo sea válido.

## <a name="syntax"></a>Sintaxis

> ⟦*Name*⟧ **QWord** *initializer* ⟦ __,__ *initializer* ... ⟧
