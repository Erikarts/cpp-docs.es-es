# <a name="operator--"></a>\- (Operador)

Devuelve *expression1* menos *expresión2*. El segundo operador invierte el signo de la *expresión*.

## <a name="syntax"></a>Sintaxis

> *expression1* **-** *expression2*
>
> __-__ *expresión* )
