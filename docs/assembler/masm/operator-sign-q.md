# <a name="operator-sign"></a>Operador SIGN?

Estado de la marca de signo. Se usa solo dentro de [. Si](../../assembler/masm/dot-if.md)es, [. WHILE](../../assembler/masm/dot-while.md), o [. REPETIR](../../assembler/masm/dot-repeat.md) bloques y evaluarlos en tiempo de ejecución, no en tiempo de ensamblado.

## <a name="syntax"></a>Sintaxis

> **SESIÓN?**
