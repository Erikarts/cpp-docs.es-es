# <a name="defaultbind"></a>defaultbind

Indica la única propiedad enlazable que mejor representa el objeto.

## <a name="syntax"></a>Sintaxis

```cpp
[defaultbind]
```

## <a name="remarks"></a>Comentarios

El atributo **defaultbind** C++ tiene la misma funcionalidad que el atributo MIDL [defaultbind](/windows/win32/Midl/defaultbind) .

## <a name="example"></a>Ejemplo

Vea el ejemplo de [Bindable](bindable.md) para ver un ejemplo de cómo usar **defaultbind**.
