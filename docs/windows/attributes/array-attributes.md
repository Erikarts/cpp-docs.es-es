# <a name="array-attributes"></a>Atributos de matriz

Los siguientes atributos se aplican a las matrices o `SAFEARRAY`s.

|Atributo|Descripción|
|---------------|-----------------|
|[library_block](library-block.md)|Coloca una construcción dentro del bloque de biblioteca del archivo. idl.|
|[satype](satype.md)|Especifica el tipo de datos de la estructura de `SAFEARRAY`.|
|[string](string-cpp.md)|Indica que la matriz **Char**, **wchar_t**, `byte` (o equivalente) unidimensional o el puntero a dicha matriz se deben tratar como una cadena.|
