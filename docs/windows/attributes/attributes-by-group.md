# <a name="attributes-by-group"></a>Atributos por grupo

Los C++ atributos se organizan en los siguientes grupos funcionales.

|Atributo|Descripción|
|---------------|-----------------|
|[Atributos COM](com-attributes.md)|Inserte código para admitir numerosas áreas de desarrollo COM y .NET Framework Common Language Runtime desarrollo.|
|[Atributos IDL](idl-attributes.md)|Habilite la modificación del archivo. idl desde un archivo de código fuente sin el uso de un asistente y sin conocimiento de la estructura y la sintaxis de dicho archivo.|
|[Atributos de consumidor OLE DB](ole-db-consumer-attributes.md)|Inserte el código, basado en las [plantillas de consumidor OLE DB](../../data/oledb/ole-db-consumer-templates-reference.md), para crear un consumidor OLE DB que realice tareas como abrir tablas, ejecutar comandos y obtener acceso a los datos.|
|[Atributos de compilador](compiler-attributes.md)|Proporcionado por el compilador de Microsoft C++ .|

Para obtener información sobre cómo definir atributos personalizados para la aplicación .NET Framework, vea [atributos definidos por el usuario](../../extensions/user-defined-attributes-cpp-component-extensions.md).
