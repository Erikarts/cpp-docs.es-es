# <a name="regex_traitsltchargt-class"></a>regex_traits&lt;char&gt; (Clase)

Especialización de `regex_traits` para **Char**.

## <a name="syntax"></a>Sintaxis

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>Comentarios

La clase es una especialización explícita de la plantilla de clase [regex_traits](../standard-library/regex-traits-class.md) para los elementos de tipo **Char** (para que pueda aprovechar las funciones de biblioteca que manipulan objetos de este tipo).

## <a name="requirements"></a>Requisitos

**Encabezado:** \<regex>

**Espacio de nombres:** std
