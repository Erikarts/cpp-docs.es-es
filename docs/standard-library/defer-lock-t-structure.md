# <a name="deferlockt-structure"></a>defer_lock_t (Estructura)

Representa un tipo que define un objeto [defer_lock](../standard-library/mutex-functions.md#defer_lock) que se usa para seleccionar uno de los constructores sobrecargados de [unique_lock](../standard-library/unique-lock-class.md).

## <a name="syntax"></a>Sintaxis

```cpp
struct defer_lock_t;
```

## <a name="requirements"></a>Requisitos

**Encabezado:** \<exclusión mutua >

**Espacio de nombres:** std
