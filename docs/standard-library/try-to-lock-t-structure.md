# <a name="trytolockt-structure"></a>try_to_lock_t (Estructura)

Representa un **struct** que define un objeto [try_to_lock](../standard-library/mutex-functions.md#try_to_lock) . Se usa para seleccionar uno de los constructores sobrecargados de [unique_lock](../standard-library/unique-lock-class.md).

## <a name="syntax"></a>Sintaxis

```cpp
struct try_to_lock_t;
```

## <a name="requirements"></a>Requisitos

**Encabezado:** \<exclusión mutua >

**Espacio de nombres:** std
