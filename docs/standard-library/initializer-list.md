# <a name="ltinitializer_listgt"></a>&lt;initializer_list&gt;

Define la plantilla de clase contenedora `initializer_list` y varias plantillas auxiliares.

## <a name="syntax"></a>Sintaxis

```cpp
#include <initializer_list>
```

### <a name="classes"></a>Clases

|Clase|Descripción|
|-|-|
|[initializer_list](../standard-library/initializer-list-class.md)|Proporciona acceso a una matriz de elementos en la que cada miembro es del tipo especificado.|
