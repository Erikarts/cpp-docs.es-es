# <a name="ltsample-containergt"></a>&lt;contenedor de ejemplo&gt;

> [!NOTE]
> Este tema se encuentra en la documentación de Microsoft C++ como un ejemplo no funcional de contenedores utilizados en la biblioteca estándar de C++. Para más información, vea [Contenedores de la biblioteca estándar de C++](../standard-library/stl-containers.md).

Muestra la estructura de los encabezados de contenedor en la biblioteca estándar de C++.

## <a name="syntax"></a>Sintaxis

```cpp
#include <sample container> // Nonfunctional header
```
