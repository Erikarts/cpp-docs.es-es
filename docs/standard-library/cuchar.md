# <a name="ltcwchargt"></a>&lt;cwchar&gt;

Incluye el encabezado de la biblioteca estándar de C \<uchar.h > y agrega los nombres asociados a la `std` espacio de nombres.

## <a name="requirements"></a>Requisitos

**Encabezado:** \<cuchar >

**Espacio de nombres:** std

## <a name="member-functions"></a>Funciones miembro

|||
|-|-|
|[c16rtomb](../standard-library/cuchar-functions.md#c16rtomb)|Sin descripción.|
|[c32rtomb](../standard-library/cuchar-functions.md#c32rtomb)|Sin descripción.|
|[mbrtoc16](../standard-library/cuchar-functions.md#mbrtoc16)|Sin descripción.|
|[mbrtoc32](../standard-library/cuchar-functions.md#mbrtoc32)|Sin descripción.|

## <a name="remarks"></a>Comentarios

Incluir este encabezado también garantiza que los nombres declarados mediante vinculación externa en el encabezado de la biblioteca estándar de C se declaran en el espacio de nombres `std`.
