# <a name="regextraitsltwchartgt-class"></a>regex_traits&lt;wchar_t&gt; (Clase)

Especialización de `regex_traits` para **wchar_t**.

## <a name="syntax"></a>Sintaxis

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>Comentarios

La clase es una especialización explícita de la clase de plantilla [regex_traits](../standard-library/regex-traits-class.md) para los elementos de tipo **wchar_t** (para que pueda aprovechar las funciones de la biblioteca que manipulan objetos de este tipo).

## <a name="requirements"></a>Requisitos

**Encabezado:** \<regex>

**Espacio de nombres:** std
