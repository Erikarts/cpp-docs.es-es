# <a name="ltcvtwstringgt"></a>&lt;cvt/wstring&gt;

El encabezado `<cvt/wstring>` en versiones anteriores de Visual Studio definía la [clase wstring_convert](../standard-library/wstring-convert-class.md) de la plantilla de clase en el espacio de nombres stdext:: CVT. El encabezado se conserva por razones de compatibilidad con las versiones anteriores. El código nuevo debe usar la versión de la clase que se define en [\<locale>](../standard-library/locale.md), en el espacio de nombres `std`.

## <a name="syntax"></a>Sintaxis

```cpp
#include <cvt/wstring>
```
