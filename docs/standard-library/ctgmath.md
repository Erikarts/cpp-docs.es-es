# <a name="ltctgmathgt"></a>&lt;ctgmath&gt;

En efecto, incluye el C++ encabezados de la biblioteca estándar \<complejo > y \<cmath >, que proporcionan las macros math genéricas de tipo equivalente a \<tgmath.h >.

## <a name="requirements"></a>Requisitos

**Encabezado:** \<ctgmath >

**Espacio de nombres:** std

## <a name="remarks"></a>Comentarios

La funcionalidad del encabezado de biblioteca estándar de C \<tgmath.h > proporcionan las sobrecargas en \<complejo > y \<cmath >.

Incluir este encabezado también garantiza que los nombres declarados mediante vinculación externa en el encabezado de la biblioteca estándar de C se declaran en el espacio de nombres `std`.
