# <a name="compiler-error-c3818"></a>Error del compilador C3818

declaración de propiedad de matriz 'Propiedad1' no debe sobrecargar una propiedad index 'property2'

Una sobrecarga no es posible que las propiedades cuando uno es un indizador y el otro es una propiedad de matriz.

Solo es accesible a través de la opción del compilador obsoleto C3818 **/CLR: oldSyntax**.
