# <a name="compiler-error-c2647"></a>Error del compilador C2647

' Operator ': no se puede desreferenciar un ' tipo1 ' en un ' tipo2 '

El operando izquierdo de un operador de puntero a miembro (`->*` o `.*`) no se puede convertir implÃ­citamente en un tipo relacionado con el operador derecho.

En el ejemplo siguiente se genera C2647:

```cpp
// C2647.cpp
class C {};
class D {};

int main() {
   D d, *pd;
   C c, *pc = 0;
   int C::*pmc = 0;
   pd->*pmc = 0;   // C2647
   d.*pmc = 0;   // C2647

   // OK
   pc->*pmc = 0;
   c.*pmc = 0;
}
```
