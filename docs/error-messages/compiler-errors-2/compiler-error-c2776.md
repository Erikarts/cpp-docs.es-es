# <a name="compiler-error-c2776"></a>Error del compilador C2776

solo se puede especificar un método ' get ' por propiedad

Solo puede especificar una función `get` en el atributo extendido [Property](../../cpp/property-cpp.md) . Este error se produce cuando se especifican varias funciones de `get`.

En el ejemplo siguiente se genera C2776:

```cpp
// C2776.cpp
struct A {
   __declspec(property(get=GetProp,get=GetPropToo))
   // try the following line instead
   // __declspec(property(get=GetProp))
      int prop;   // C2776
   int GetProp(void);
   int GetPropToo(void);
};
```
