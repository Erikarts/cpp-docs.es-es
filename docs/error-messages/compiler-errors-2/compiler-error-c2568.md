# <a name="compiler-error-c2568"></a>Error del compilador C2568

' identificador1 ': no se puede resolver la sobrecarga de la función

El compilador no puede determinar la función sobrecargada a la que se va a llamar. Los parámetros reales que se pasan a la función deben convertirse para que coincidan con los parámetros formales de una de las funciones sobrecargadas, pero ninguna de las coincidencias es una mejor manera inequívoca que las demás.
