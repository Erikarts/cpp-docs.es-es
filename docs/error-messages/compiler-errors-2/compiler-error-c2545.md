# <a name="compiler-error-c2545"></a>Error del compilador C2545

' Operator ': no se puede encontrar el operador sobrecargado

No se puede usar el operador con los operandos proporcionados. Debe proporcionar un operador sobrecargado con los operandos necesarios.

Este error puede deberse a que los operandos tienen un tipo incorrecto.

Este error puede corregirse si se define un operador de conversión o un constructor que toma un único parámetro.
