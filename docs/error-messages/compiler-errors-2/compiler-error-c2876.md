# <a name="compiler-error-c2876"></a>Error del compilador C2876

' Class:: Symbol ': no se puede tener acceso a todas las sobrecargas

Todas las formas sobrecargadas de una funci√≥n de una clase base deben ser accesibles para la clase derivada.

En el ejemplo siguiente se genera C2876:

```cpp
// C2876.cpp
// compile with: /c
class A {
public:
   double a(double);
private:
   int a(int);
};

class B : public A {
   using A::a;   // C2876 one overload is private in base class
};
```
