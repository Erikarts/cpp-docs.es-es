# <a name="compiler-error-c2713"></a>Error del compilador C2713

solo se permite una forma de control de excepciones por función

No se puede utilizar el control de excepciones estructurado (`__try`/C++ `__except`) y el control de excepciones (`try`/`catch`) en la misma función.
