# <a name="compiler-error-c2948"></a>Error del compilador C2948

creación de instancias explícita; no se permite el especificador de clase de almacenamiento ' especificador ' en la especialización

No se pueden usar especificadores de clase de almacenamiento (como `extern`) en una especialización de una clase de plantilla de la que se ha creado previamente una instancia explícita.
