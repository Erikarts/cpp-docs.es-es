# <a name="compiler-error-c2612"></a>Error del compilador C2612

' char ' final no válido en la lista de inicializadores de base o miembro

Aparece un carácter después de la última base o miembro de una lista de inicializadores.

En el ejemplo siguiente se genera C2612:

```cpp
// C2612.cpp
class A {
public:
   int i;
   A( int ia ) : i( ia ) + {};   // C2612
};
```
