# <a name="compiler-error-c3333"></a>Error del compilador C3333

'biblioteca de tipos': una biblioteca de tipos dañada no se puede importar (#import).

El compilador no puede leer la biblioteca de tipos especificada en la instrucción `#import` . Puede volver a generar la biblioteca de tipos, si es posible, o pedir una nueva a su proveedor. Puede usar el Visor de OLE suministrado con Visual C++ para ver el archivo de biblioteca de tipos y saber lo que le pasa.
