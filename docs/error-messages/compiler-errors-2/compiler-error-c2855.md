# <a name="compiler-error-c2855"></a>Error del compilador C2855

opción de línea de comandos ' opción ' incoherente con el encabezado precompilado

La opción de línea de comandos especificada difiere de la opción utilizada para crear el encabezado precompilado. Vuelve a generar el encabezado precompilado con la opción de línea de comandos especificada.
