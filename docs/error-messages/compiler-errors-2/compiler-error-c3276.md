# <a name="compiler-error-c3276"></a>Error del compilador C3276

'keyword': salto fuera del bloque __finally/finally tiene un comportamiento indefinido durante el control de finalización

Este error es el mismo que el [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) advertencia. Sin embargo, cuando se usa/CLR, esta condición no se puede deshabilitar con la [advertencia](../../preprocessor/warning.md) pragma.
