# <a name="compiler-error-c3192"></a>Error del compilador C3192

error de sintaxis: ' ^ ' no es un operador de prefijo (Â¿quiso decir ' * '?)

No se puede usar un identificador como operador de desreferencia.

En el ejemplo siguiente se genera C3192:

```cpp
// C3192.cpp
// compile with: /clr
using namespace System;

ref class MyClass {
public:
   MyClass () {}
   MyClass(MyClass%) {}
};

int main() {
   MyClass ^ s = gcnew MyClass;
   MyClass b = ^s;   // C3192

   // OK
   MyClass b2 = *s;
}
```
