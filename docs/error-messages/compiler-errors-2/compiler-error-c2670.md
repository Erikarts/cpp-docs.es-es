# <a name="compiler-error-c2670"></a>Error del compilador C2670

' Identifier ': la plantilla de función no puede convertir el número de parámetro del tipo ' type '

No se pudo convertir un parámetro al tipo requerido.

Este error puede corregirse si se crea una conversión explícita.
