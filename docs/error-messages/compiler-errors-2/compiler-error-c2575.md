# <a name="compiler-error-c2575"></a>Error del compilador C2575

' Identifier ': solo las funciones miembro y las bases pueden ser virtuales

Se declara una función o una clase global `virtual`. Esto no está permitido.

En el ejemplo siguiente se genera C2575:

```cpp
// C2575.cpp
// compile with: /c
virtual void func() {}   // C2575

void func2() {}
struct A {
   virtual void func2(){}
};
```
