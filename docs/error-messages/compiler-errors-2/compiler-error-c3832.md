# <a name="compiler-error-c3832"></a>Error del compilador C3832

' Biblioteca de tipos ': la biblioteca de tipos parece si se compiló para punteros de 32 bits; Cambie el calificador "ptrsize".

La información explícita suministrada con el `ptrsize` atributo de la Directiva de [#import](../../preprocessor/hash-import-directive-cpp.md) no coincidió con lo que el compilador encontró en la biblioteca de tipos.
