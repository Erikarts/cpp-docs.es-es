# <a name="compiler-error-c2811"></a>Error del compilador C2811

' tipo1 ': no se puede heredar de ' tipo2 ', una clase Ref solo puede heredar de una clase Ref o de una clase de interfaz

Ha intentado usar una clase no administrada como una clase base para una clase administrada.

En el ejemplo siguiente se genera C2811:

```cpp
// C2811.cpp
// compile with: /clr /c
struct S{};
ref struct T {};
ref class C : public S {};   // C2811
ref class D : public T {};   // OK
```
