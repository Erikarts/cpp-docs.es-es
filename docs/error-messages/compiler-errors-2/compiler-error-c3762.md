# <a name="compiler-error-c3762"></a>Error del compilador C3762

no se puede procesar el atributo ' atributo '

Un atributo definido por el usuario que hereda de `System.Security.Permissions.SecurityAttribute` se utiliza para definir un atributo de seguridad. Este tipo de atributo no se puede usar en el mismo ensamblado en el que se define.
