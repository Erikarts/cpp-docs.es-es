# <a name="compiler-error-c2502"></a>Error del compilador C2502

' Identifier ': hay demasiados modificadores de acceso en la clase base

La clase base tiene m√°s de un modificador de acceso. Solo se permite un modificador de acceso (`public`, `private`o `protected`).

En el ejemplo siguiente se genera C2502:

```cpp
// C2502.cpp
// compile with: /c
class A { };
class B { };
class C : private public A { };   // C2502

// OK
class D : private A {};
class E : public A, private B {};
```
