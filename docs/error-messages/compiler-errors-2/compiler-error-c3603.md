# <a name="compiler-error-c3603"></a>Error del compilador C3603

' Symbol ': el tipo ' type ' todavía no se admite

Ha intentado usar un tipo de datos que aún no es compatible con el tiempo de ejecución de .NET en un objeto administrado.
