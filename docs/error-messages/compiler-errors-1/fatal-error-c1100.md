# <a name="fatal-error-c1100"></a>Error irrecuperable C1100

no se puede inicializar OLE: mensaje de error de sistema

El compilador no puede inicializar la biblioteca COM (modelo de objetos componentes). Consulte [CoInitialize](/windows/win32/api/objbase/nf-objbase-coinitialize).
