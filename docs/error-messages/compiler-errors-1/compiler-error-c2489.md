# <a name="compiler-error-c2489"></a>Error del compilador C2489

' Identifier ': la variable auto o Register inicializada no se permite en el ámbito de la función en la función ' naked '

Para obtener más información, vea [naked](../../cpp/naked-cpp.md).

En el ejemplo siguiente se genera C2489:

```cpp
// C2489.cpp
// processor: x86
__declspec( naked ) int func() {
   int i = 1;   // C2489
   register int j = 1;   // C2489
}
```
