# <a name="compiler-error-c2308"></a>Error del compilador C2308

concatenación de cadenas no coincidentes

Se han especificado cadenas de caracteres anchos y no anchos para la concatenación. No se puede concatenar una cadena de caracteres anchos y una cadena de caracteres no anchos.
