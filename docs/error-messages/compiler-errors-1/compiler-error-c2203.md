# <a name="compiler-error-c2203"></a>Error del compilador C2203

el operador delete no puede especificar límites para una matriz

Con la opción **/za** (ANSI), el operador `delete` puede eliminar una matriz completa, pero no partes o miembros específicos de la matriz.

En el ejemplo siguiente se genera C2203:

```cpp
// C2203.cpp
// compile with: /Za
int main() {
   int *ar = new int[10];
   delete [4] ar;   // C2203
   // try the following line instead
   // delete [] ar;
}
```
