# <a name="compiler-error-c2412"></a>Error del compilador C2412

' etiqueta ': se ha redefinido la etiqueta sin distinción de mayúsculas y minúsculas

La etiqueta se define más de una vez en la función actual. Cambie la ortografía de la etiqueta y sus referencias.
