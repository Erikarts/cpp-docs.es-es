# <a name="fatal-error-c1051"></a>Error irrecuperable C1051

el archivo de base de datos de programa, ' pdbfile ', tiene un formato obsoleto, elimínelo y vuelva a compilar

El compilador no puede actualizar el archivo de base de datos de programa, que tiene un número de versión anterior. Elimine el archivo y vuelva a compilar el programa con **/Zi** o **/Zi**. Para obtener más información, vea [/Z7,/Zi,/Zi (formato de la información de depuración)](../../build/reference/z7-zi-zi-debug-information-format.md) .
