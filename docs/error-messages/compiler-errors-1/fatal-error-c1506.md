# <a name="fatal-error-c1506"></a>Error irrecuperable C1506

error irrecuperable de ámbito de bloque

El bloque era demasiado grande para compilarlo.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Posibles causas del error:

1. Llaves no coincidentes

1. Función o clase inusualmente grande
