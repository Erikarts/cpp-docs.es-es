# <a name="fatal-error-c1196"></a>Error irrecuperable C1196

'identifier': el identificador encontrado en la biblioteca de tipos 'typelib' no es un identificador de C++ válido.

Uno de los identificadores encontrado en la biblioteca de tipos no es un identificador de C++ válido. La biblioteca de tipos no está disponible para su uso con [#import](../../preprocessor/hash-import-directive-cpp.md).
