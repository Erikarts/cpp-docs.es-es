# <a name="fatal-error-c1057"></a>Error irrecuperable C1057

final de archivo inesperado en la expansión de macros

El compilador alcanzó el final del archivo de código fuente mientras se recopilaban argumentos de invocación de macros, probablemente debido a que falta un paréntesis de cierre en la invocación de macro.
