# <a name="compiler-error-c2410"></a>Error del compilador C2410

' Identifier ': nombre de miembro ambiguo en ' context '

El identificador es un miembro de más de una estructura o unión en este contexto.

Use un especificador de estructura o de Unión en el operando que causó el error. Un especificador de estructura o unión es un identificador de tipo `struct` o `union` (un nombre de `typedef` o una variable del mismo tipo que la estructura o unión a la que se hace referencia). El especificador debe ser el operando izquierdo del primer operador de selección de miembro (.) para usar el operando.
