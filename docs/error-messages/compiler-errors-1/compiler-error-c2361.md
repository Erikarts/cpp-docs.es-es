# <a name="compiler-error-c2361"></a>Error del compilador C2361

inicialización de 'identificador' se omite en la etiqueta 'default'

La inicialización de `identifier` puede omitirse en un `switch` instrucción. No se puede saltar una declaración con un inicializador a menos que la declaración está incluida en un bloque. (A menos que se declara dentro de un bloque, la variable está dentro del ámbito hasta el final de la `switch` instrucción.)

El ejemplo siguiente genera C2361:

```
// C2361.cpp
void func( void ) {
   int x;
   switch (x) {
   case 0 :
      int i = 1;
      { int j = 1; }
   default :   // C2361 error
      int k = 1;
   }
}
```

Posible resolución:

```
// C2361b.cpp
// compile with: /c
void func( void ) {
   int x = 0;
   switch (x) {
   case 0 :
      { int j = 1; int i = 1;}
   default :
      int k = 1;
   }
}
```
