# <a name="fatal-error-c1046"></a>Error irrecuperable C1046

límite del compilador: la estructura está demasiado anidada

La estructura, Unión o clase superó el límite de anidamiento, que es 15 niveles. Vuelva a escribir la definición para reducir el nivel de anidamiento. Divida la estructura, Unión o clase en dos o más partes mediante `typedef` para definir una o varias de las estructuras anidadas.
