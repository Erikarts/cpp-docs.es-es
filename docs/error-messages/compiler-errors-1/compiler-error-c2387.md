# <a name="compiler-error-c2387"></a>Error del compilador C2387

' type ': clase base ambigua

El compilador no pudo resolver de forma inequívoca una llamada de función porque la función existe en más de una clase base.

Para resolver este error, quite una de las clases base de la herencia, o bien califique explícitamente la llamada de función.

En el ejemplo siguiente se genera C2387:

```cpp
// C2387.cpp
namespace N1 {
   struct B {
      virtual void f() {
      }
   };
}

namespace N2 {
   struct B {
      virtual void f() {
      }
   };
}

struct D : N1::B, N2::B {
   virtual void f() {
      B::f();   // C2387
      // try the following line instead
      // N1::B::f();
   }
};

int main() {
   D aD;
   aD.f();
}
```
