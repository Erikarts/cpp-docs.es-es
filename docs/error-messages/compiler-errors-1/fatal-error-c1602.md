# <a name="fatal-error-c1602"></a>Error irrecuperable C1602

intrínseco no admitido

Este error indica una falta de coincidencia en los archivos. exe del compilador, que pueden haberse producido debido a una instalación incompleta. Por ejemplo, es posible que haya instalado una Service Pack pero no el paquete de procesadores. Instale todos los productos necesarios.
