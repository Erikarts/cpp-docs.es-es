# <a name="fatal-error-c1603"></a>Error irrecuperable C1603

destino de bifurcación de ensamblado alineado fuera del intervalo por ' número ' bytes

La distancia calculada entre una instrucción JCXZ o JECXZ y su etiqueta de destino especificada era superior a 128 bytes. Actualice el código para que la etiqueta esté más cerca de la instrucción.
