# <a name="compiler-error-c2018"></a>Error del compilador C2018

carácter 'hexnumber' desconocido

El archivo de código fuente contiene un carácter ASCII inesperado, que se identifica por su número hexadecimal. Para resolver el error, quite el carácter.
