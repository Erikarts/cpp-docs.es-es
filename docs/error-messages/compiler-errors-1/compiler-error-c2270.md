# <a name="compiler-error-c2270"></a>Error del compilador C2270

' función ': modificadores no permitidos en funciones no miembro

Una función no miembro se declara con [const](../../cpp/const-cpp.md), [volatile](../../cpp/volatile-cpp.md)u otro modificador de modelo de memoria.

En el ejemplo siguiente se genera C2270:

```cpp
// C2270.cpp
// compile with: /c
void func1(void) const;   // C2270, nonmember function

void func2(void);

class CMyClass {
public:
   void func2(void) const;
};
```
