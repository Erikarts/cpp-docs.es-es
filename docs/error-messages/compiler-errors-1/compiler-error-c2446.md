# <a name="compiler-error-c2446"></a>Error del compilador C2446

' Operator ': no hay conversión de ' tipo1 ' a ' tipo2 '

El compilador no puede convertir `type1` en `type2`. Es posible que la conversión no tenga sentido porque infringe la semánticaC++ C/.
