# <a name="fatal-error-c1108"></a>Error irrecuperable C1108

> no se puede encontrar la DLL: '*nombre de dll*'

No se encontró el archivo DLL especificado (*nombre de dll*) en la ruta de acceso.

Para resolver este error, vuelva a instalar C++ visualmente o copie el archivo. dll adecuado de la instalación de en el equipo.
