# <a name="fatal-error-c1904"></a>Error irrecuperable C1904

interacción con el proveedor incorrecta: 'file'

Este error indica un error de un proveedor de atributos.

Anote las circunstancias del error, intente aislar el problema y crear un caso de prueba reproducible y, a continuación, póngase en contacto con [nosotros](/visualstudio/ide/talk-to-us).
