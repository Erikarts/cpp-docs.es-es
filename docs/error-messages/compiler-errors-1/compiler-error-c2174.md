# <a name="compiler-error-c2174"></a>Error del compilador C2174

' función ': el parámetro real tiene el tipo ' void ': parámetro número1, lista de parámetros número2

`number1` de parámetros pasados a la lista de parámetros `number2` es un parámetro de `void`. Los parámetros no pueden tener el tipo `void`. En su lugar, use `void*`.
