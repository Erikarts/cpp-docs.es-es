# <a name="fatal-error-c1107"></a>Error irrecuperable C1107

no se pudo encontrar el ensamblado ' archivo ': especifique la ruta de acceso de búsqueda del ensamblado utilizando/AI o estableciendo la variable de entorno LIBPATH.

Se pasó un archivo de metadatos a una directiva de [#using](../../preprocessor/hash-using-directive-cpp.md) que el compilador no pudo encontrar.

LIBPATH, que se describe en el tema para obtener `#using`, y la opción [/AI](../../build/reference/ai-specify-metadata-directories.md) del compilador le permite especificar los directorios en los que el compilador buscará los archivos de metadatos a los que se hace referencia.
