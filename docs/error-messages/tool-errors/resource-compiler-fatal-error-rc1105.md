# <a name="resource-compiler-fatal-error-rc1105"></a>Error irrecuperable del compilador de recursos RC1105

modificador no válido, opción: demasiados modificadores/d

Se definieron demasiados símbolos mediante la opción de definición de símbolos (/d). Defina algunos símbolos en el origen y vuelva a compilar.
