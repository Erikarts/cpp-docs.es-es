# <a name="expression-evaluator-error-cxx0038"></a>Error del evaluador de expresiones CXX0038

número de argumentos de función y/o error de coincidencia de tipos

La llamada de función tal y como se especificó no coincide con el prototipo de la función.

Vuelva a escribir la llamada con el número correcto de argumentos. Tipo convierta cada argumento para que coincida con el prototipo, según sea necesario.

Este error es idéntico a CAN0038.
