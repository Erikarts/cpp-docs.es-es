# <a name="project-build-error-prj0050"></a>Error PRJ0050 al compilar el proyecto

No se pudo registrar la salida. Asegúrese de que tiene los permisos adecuados para modificar el registro.

El sistema C++ de compilación visual no pudo registrar el resultado de la compilación (dll o. exe). Debe haber iniciado sesión como administrador para modificar el registro.

Si va a compilar un archivo. dll, puede intentar registrar el archivo. dll manualmente mediante regsvr32. exe, que debe mostrar información sobre por qué se produjo un error en la compilación.

Si no va a compilar un archivo. dll, busque el comando que produce un error en el registro de compilación.
