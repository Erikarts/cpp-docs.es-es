# <a name="expression-evaluator-error-cxx0051"></a>Error del evaluador de expresiones CXX0051

se especificó más de un símbolo sobrecargado en el punto de interrupción

El evaluador de expresiones no pudo determinar cuál de los símbolos sobrecargados se utiliza como punto de interrupción.

Este error es idéntico a CAN0051.
