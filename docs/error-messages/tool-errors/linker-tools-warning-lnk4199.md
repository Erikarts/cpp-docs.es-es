# <a name="linker-tools-warning-lnk4199"></a>Advertencia de las herramientas del vinculador LNK4199

Se ha omitido/DELAYLOAD: DllName; no se encontraron importaciones de DllName

El enlazador omite `dllname` porque no necesita ninguna de las funciones que `dllname` exporta.
