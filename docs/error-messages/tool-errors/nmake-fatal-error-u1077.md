# <a name="nmake-fatal-error-u1077"></a>Error grave de NMAKE U1077

' Program ': código de retorno ' value '

El comando o programa en cuestión al que ha llamado NMAKE ha producido un error y ha devuelto el código de salida proporcionado.

Para suprimir este error y continuar con la sesión de NMAKE, use la opción/I, **. IGNORE** la Directiva DOT o el modificador de comando Dash ( **-** ). Para continuar con la sesión de NMAKE de partes no relacionadas del árbol de dependencias, use la opción /K.
