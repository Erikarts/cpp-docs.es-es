# <a name="nmake-fatal-error-u1087"></a>Error grave de NMAKE U1087

no puede tener: y:: dependientes para el mismo destino

No se puede especificar un destino en un carácter de dos puntos ( **:** ) y una dependencia de dos puntos (`::`).

Para especificar un destino en varios bloques de descripción, use `::` en cada línea de dependencia.
