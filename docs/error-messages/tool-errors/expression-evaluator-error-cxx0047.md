# <a name="expression-evaluator-error-cxx0047"></a>Error del evaluador de expresiones CXX0047

la lista de argumentos no coincide con una función

Una expresión llamada una función con una lista de parámetros real que no coincidía con la lista de parámetros formales de cualquier función con el mismo nombre definido en el programa.

Solo se puede llamar a las funciones sobrecargadas si hay una coincidencia de parámetro exacta o una coincidencia que no requiere la construcción de un objeto.

Este error es idéntico a CAN0047.
