# <a name="nmake-warning-u4001"></a>Advertencia de NMAKE U4001

> el archivo de comandos solo se puede invocar desde la línea de comandos

Un archivo de comandos, que es invocado por el especificador arroba ( **\@** ), no puede contener una especificación para otro archivo de comandos. Este anidamiento no está permitido. Se omitió la especificación.
