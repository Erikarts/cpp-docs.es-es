# <a name="linker-tools-error-lnk1141"></a>Error de las herramientas del vinculador LNK1141

error durante la compilación del archivo de exportaciones

[Link](../../build/reference/linking.md) no pudo generar el archivo de exportaciones (. exp).

Entre las causas de este error se incluyen problemas del sistema, como memoria insuficiente y errores de sintaxis en las opciones o en las instrucciones de definición de módulos.

Este error está precedido por otro error que proporciona información adicional.
