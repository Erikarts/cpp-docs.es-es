# <a name="linker-tools-error-lnk1189"></a>Error de las herramientas del vinculador LNK1189

LIBTOOMANYMEMBERS:: límite de la biblioteca de objetos excedido

Se superó el límite de 65535 objetos o miembros de una biblioteca.

Para corregir este problema, divida la biblioteca en dos o más bibliotecas con menos objetos o miembros.
