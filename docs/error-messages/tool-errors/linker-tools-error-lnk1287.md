# <a name="linker-tools-error-lnk1287"></a>Error de las herramientas del vinculador LNK1287

función de punto de entrada administrada no válida

El punto de entrada no es válido para una imagen administrada. El tipo de valor devuelto de una función de punto de entrada administrado solo se puede `void` o `int`. El tipo de parámetro de una función de punto de entrada administrado solo se puede `void` o `String []`.
