# <a name="resource-compiler-error-rc2167"></a>Error del compilador de recursos RC2167

campo VERSIONINFO no reconocido; Se esperaba una coma o inicio

Se encontró un campo no reconocido en el **FIXED** forma parte de un **VERSIONINFO** declaración de estructura. Un **VERSIONINFO** campo debe ser DWORD separado por comas.
