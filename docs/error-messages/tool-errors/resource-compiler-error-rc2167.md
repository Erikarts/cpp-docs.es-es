# <a name="resource-compiler-error-rc2167"></a>Error del compilador de recursos RC2167

campo VERSIONINFO no reconocido; Se esperaba BEGIN o una coma

Se encontró un campo no reconocido en la parte **fija** de una declaración de estructura **versionInfo** . Un campo **versionInfo** debe ser un valor DWORD separado por una coma.
