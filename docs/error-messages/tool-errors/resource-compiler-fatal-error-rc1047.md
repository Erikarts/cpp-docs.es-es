# <a name="resource-compiler-fatal-error-rc1047"></a>Error irrecuperable del compilador de recursos RC1047

"demasiadas opciones de opción, ' cadena '"

La opción especificada se especificó demasiadas veces. La cadena especificada es el argumento de la opción que causó el error.
