# <a name="expression-evaluator-error-cxx0028"></a>Error del evaluador de expresiones CXX0028

elemento no struct/union

Se especificó una expresión con el formato ' struct. Member ' o ' pStruct-> Member ', pero \<miembro > no es un elemento de la estructura.

Es posible que la expresión no se incluya correctamente entre paréntesis.

Este error es idéntico a CAN0028.
