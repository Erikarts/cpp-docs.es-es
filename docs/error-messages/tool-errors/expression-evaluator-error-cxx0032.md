# <a name="expression-evaluator-error-cxx0032"></a>Error del evaluador de expresiones CXX0032

dividir por 0

La expresión contiene un divisor de cero, que no es válido. Este divisor puede ser el número literal cero, o puede ser una expresión que se evalúe como cero.

Este error es idéntico a CAN0032.
