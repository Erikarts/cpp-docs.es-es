# <a name="project-build-error-prj0046"></a>Error PRJ0046 al compilar el proyecto

No se pudo generar la línea de comandos porque la especificada estaba vacía.

Se especificó una línea de comandos vacía para una configuración de archivos make; la línea de comandos es obligatoria.
