# <a name="linker-tools-error-lnk1113"></a>Error de las herramientas del vinculador LNK1113

tipo de equipo no válido

El tipo de máquina especificado en el encabezado de objeto no es válido.

Este error puede deberse a que el archivo está dañado. Vuelva a generar el objeto.
