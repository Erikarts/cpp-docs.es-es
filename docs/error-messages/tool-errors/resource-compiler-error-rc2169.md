# <a name="resource-compiler-error-rc2169"></a>Error del compilador de recursos RC2169

el nombre del archivo de recursos no tiene el formato 2,03

El recurso especificado usó un formato anterior a la versión 2,03. El archivo de recursos se debe convertir o volver a crear con el formato de la versión 3,00 o posterior.
