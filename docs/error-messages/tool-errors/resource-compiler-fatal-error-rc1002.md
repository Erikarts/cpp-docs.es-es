# <a name="resource-compiler-fatal-error-rc1002"></a>Error irrecuperable del compilador de recursos RC1002

espacio de montón insuficiente

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Para corregir mediante las siguientes posibles soluciones

1. Aumente el espacio de archivo de intercambio de Windows. Para obtener más información sobre cómo aumentar el espacio de archivo de intercambio, consulte memoria virtual en la ayuda de Windows.

1. Divida el archivo actual en archivos más pequeños y compílelo por separado.

1. Quitar otros programas o controladores que se ejecutan en el sistema.
