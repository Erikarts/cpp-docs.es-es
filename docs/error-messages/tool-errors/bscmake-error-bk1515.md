# <a name="bscmake-error-bk1515"></a>Error de BSCMAKE BK1515

bscfile: versión no compatible: no se puede actualizar incrementalmente

El archivo. BSC se creó con otra versión de BSCMAKE. Vuelva a crear el archivo. BSC.
