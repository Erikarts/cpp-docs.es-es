# <a name="linker-tools-error-lnk1136"></a>Error de las herramientas del vinculador LNK1136

archivo no válido o dañado

El archivo de entrada tiene un encabezado dañado o tiene un tamaño cero o es anormalmente pequeño.

LNK1136 también puede deberse a que el archivo se ha especificado incorrectamente como una dependencia adicional.
