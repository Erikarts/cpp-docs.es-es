# <a name="expression-evaluator-error-cxx0023"></a>Error del evaluador de expresiones CXX0023

base no válida

El evaluador de expresiones de C no reconoce la base especificada. Solo las bases decimales, hexadecimales y octales son válidas.

Este error es idéntico a CAN0023.
