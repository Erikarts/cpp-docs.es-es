# <a name="expression-evaluator-error-cxx0053"></a>Error del evaluador de expresiones CXX0053

coincidencia de símbolos que no son de función al enlazar puntos de interrupción

Un símbolo que se usa como punto de interrupción no es una función. La especificación de un miembro de datos como un punto de interrupción puede producir este error.

Este error es idéntico a CAN0053.
