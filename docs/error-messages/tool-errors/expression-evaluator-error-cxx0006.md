# <a name="expression-evaluator-error-cxx0006"></a>Error del evaluador de expresiones CXX0006

faltan paréntesis de apertura

Se encontraron paréntesis desequilibrados en la expresión. Vuelva a escribir la expresión con paréntesis equilibrados.

Este error es idéntico a CAN0006.
