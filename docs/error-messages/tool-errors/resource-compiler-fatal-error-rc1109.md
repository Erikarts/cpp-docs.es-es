# <a name="resource-compiler-fatal-error-rc1109"></a>Error irrecuperable del compilador de recursos RC1109

Error al crear el nombre de recurso

No se pudo crear el archivo. res especificado. Asegúrese de que no se está creando en una unidad de solo lectura. Use la opción/V para averiguar si se está creando el archivo.
