# <a name="linker-tools-warning-lnk4002"></a>Advertencia de las herramientas del vinculador LNK4002

símbolo definido en el objeto

El símbolo, se muestra en su forma representativa, se especificó en su formato no representativo en `object`, pero no se encontró una coincidencia única a un símbolo representativo. Esta advertencia siempre precedida de advertencia [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) y seguido por un error irrecuperable [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
