# <a name="resource-compiler-fatal-error-rc1121"></a>Error irrecuperable del compilador de recursos RC1121

Error de E/S al leer el archivo

El compilador de recursos no pudo leer un archivo. Compruebe que la unidad que contiene el archivo está disponible y que el archivo es válido.
