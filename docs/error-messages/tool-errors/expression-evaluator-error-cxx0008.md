# <a name="expression-evaluator-error-cxx0008"></a>Error del evaluador de expresiones CXX0008

**Falta "al final de cadena**

Falta la comilla doble que se esperaba el final de la cadena literal. Vuelva a escribir la expresión encerrando el literal de cadena de comillas dobles.

Este error es idéntico a CAN0008.
