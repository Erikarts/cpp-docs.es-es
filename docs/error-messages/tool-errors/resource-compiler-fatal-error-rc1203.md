# <a name="resource-compiler-fatal-error-rc1203"></a>Error irrecuperable del compilador de recursos RC1203

se especificó un ID. de idioma predeterminado hexadecimal no válido.

La opción Especificar idioma predeterminado (/l) iba seguida de un identificador de idioma hexadecimal no válido. Consulte [cadenas de idioma y de país o región](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) en la referencia de la *biblioteca en tiempo de ejecución* para obtener una lista de identificadores de idioma válidos.
