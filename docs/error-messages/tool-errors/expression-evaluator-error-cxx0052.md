# <a name="expression-evaluator-error-cxx0052"></a>Error del evaluador de expresiones CXX0052

función miembro no presente

Se especificó una función miembro como punto de interrupción, pero no se encontró. Establecer un punto de interrupción en una función que se ha insertado puede producir este error.

Vuelva a compilar el archivo con la inclusión forzada (/Ob0) para establecer un punto de interrupción en esta función.

Una expresión llamada una función que no se definió.

Este error es idéntico a CAN0052.
