# <a name="expression-evaluator-error-cxx0026"></a>Error del evaluador de expresiones CXX0026

cadena de formato incorrecto

No se especificó correctamente una cadena de formato. Cambie la sintaxis de la expresión.

Este error es idéntico a CAN0026.

Para obtener más información, vea [especificadores de C++formato en ](/visualstudio/debugger/format-specifiers-in-cpp).
