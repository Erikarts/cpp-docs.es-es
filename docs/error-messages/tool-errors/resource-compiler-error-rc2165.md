# <a name="resource-compiler-error-rc2165"></a>Error del compilador de recursos RC2165

no se encuentra la cadena en la instrucción DLGINCLUDE

La instrucción no especificó un archivo de inclusión válido.

La instrucción **DLGINCLUDE** debe utilizar la siguiente sintaxis:

**DLGINCLUDE "** <em>nombreDeArchivo</em> **. h"**
