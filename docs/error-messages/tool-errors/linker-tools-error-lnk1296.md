# <a name="linker-tools-error-lnk1296"></a>Error de las herramientas del vinculador LNK1296

no se puede cargar el nombre de archivo

La DLL especificada no estaba disponible.

Las ubicaciones posibles para el archivo DLL son el directorio actual, el directorio del sistema, el directorio de Windows y los directorios especificados en la variable de entorno PATH.
