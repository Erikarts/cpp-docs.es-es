# <a name="profile-guided-optimization-warning-pg1087"></a>Advertencia de la optimización guiada por perfiles PG1087

El archivo PGC ' file ' no es válido.

Un archivo. PGC está dañado, posiblemente debido a un extremo anómalo a una sesión de generación de perfiles.  Vuelva a ejecutar el escenario de generación de perfiles y finalice normalmente.
