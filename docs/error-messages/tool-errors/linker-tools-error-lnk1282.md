# <a name="linker-tools-error-lnk1282"></a>Error de las herramientas del vinculador LNK1282

no se puede hacer el archivo/REBASE; se ha firmado

Se intentó cambiar la dirección base de un ensamblado firmado con la opción/REBASE para [EDITBIN](../../build/reference/editbin-reference.md). Para ello, primero debe cambiar la dirección base y, a continuación, firmar el ensamblado.
