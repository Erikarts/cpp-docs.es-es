# <a name="expression-evaluator-error-cxx0025"></a>Error del evaluador de expresiones CXX0025

el operador necesita struct/union

Un operador que toma una expresión de `struct` o tipo de **Unión** se aplicó a una expresión que no es un `struct` o una **Unión**.

Los componentes de las variables de clase, estructura o Unión deben tener un nombre completo. No se pueden especificar componentes sin especificación completa.

Este error es idéntico a CAN0025.
