# <a name="expression-evaluator-error-cxx0020"></a>Error del evaluador de expresiones CXX0020

tipos de operando incorrectos para esta operación

Se aplicó un operador a una expresión con un tipo no válido para ese operador.

Por ejemplo, no es válido tomar la dirección de un registro o subíndice una matriz con una expresión de punto flotante.

Este error es idéntico a CAN0020.
