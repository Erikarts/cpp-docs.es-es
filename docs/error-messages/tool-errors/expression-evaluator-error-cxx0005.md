# <a name="expression-evaluator-error-cxx0005"></a>Error del evaluador de expresiones CXX0005

operador no admitido

Se especificó un operador de C no compatible en una expresión. Escriba una expresión equivalente con los operadores de C admitidos.

Este error es idéntico a CAN0005.
