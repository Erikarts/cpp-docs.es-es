# <a name="expression-evaluator-error-cxx0041"></a>Error del evaluador de expresiones CXX0041

el elemento Class debe ser un miembro estático o una función miembro

Se usó un miembro no estático de una clase (o una estructura o unión) sin especificar la instancia de la clase que se va a utilizar.

Solo se pueden usar miembros de datos estáticos o funciones miembro sin especificar una creación de instancias.

Este error es idéntico a CAN0041.
