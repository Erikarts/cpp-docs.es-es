# <a name="project-build-error-prj0028"></a>Error PRJ0028 al compilar el proyecto

El archivo temporal ' archivo ' contiene contenido Unicode que no se puede traducir a la página de códigos ANSI del usuario.

Se ha especificado un valor con la opción del enlazador [/MIDL (especificar opciones de línea de comandos de MIDL)](../../build/reference/midl-specify-midl-command-line-options.md) que no se pudo resolver en la página de códigos del sistema.

La página de códigos que se usa al especificar el comando MIDL (la página de códigos de entrada) debe ser la misma que la página de códigos del sistema.
