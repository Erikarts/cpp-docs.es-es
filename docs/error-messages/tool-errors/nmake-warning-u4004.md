# <a name="nmake-warning-u4004"></a>Advertencia de NMAKE U4004

demasiadas reglas para el destino ' TargetName '

Se especificó más de un bloque de descripción para el destino dado mediante dos puntos ( **:** ) como separadores. NMAKE ejecutó los comandos en el primer bloque de Descripción y omitió los bloques posteriores.

Para especificar el mismo destino en varias dependencias, utilice dos puntos dobles (`::`) como separador en cada línea de dependencia.
