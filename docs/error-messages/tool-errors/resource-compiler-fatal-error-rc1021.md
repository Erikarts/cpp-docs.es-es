# <a name="resource-compiler-fatal-error-rc1021"></a>Error irrecuperable del compilador de recursos RC1021

comando de preprocesador 'string' no válido

Los caracteres que siguen al signo de número ( **#** ) no formaban una directiva de preprocesador válida.

No se puede usar el signo de número como primer carácter de un identificador.
