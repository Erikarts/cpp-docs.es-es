# <a name="resource-compiler-error-rc2111"></a>Error del compilador de recursos RC2111

tipo de control no válido

Cada instrucción de CONTROL de una instrucción **Dialog** debe ser uno de los siguientes: 3STATE, AUTO3, autocheck, Autoradio, BEDIT, CheckBox, ComboBox, control, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, Icon, IEDIT, ListBox, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RText, ScrollBar, USERBUTTON.

Asegúrese de que estas instrucciones de CONTROL están escritas correctamente.
