# <a name="math-error-m6110"></a>Error matemático M6110

desbordamiento de pila

Una expresión de punto flotante provocó un desbordamiento de la pila de punto flotante.

Las excepciones de punto flotante de desbordamiento de pila se interceptan hasta un límite de siete niveles, además de los ocho niveles que normalmente admite el coprocesador 8087/287/387.

El programa finaliza con el código de salida 138.
