# <a name="project-build-warning-prj0018"></a>Advertencia PRJ0018 al compilar el proyecto

No se encontraron las siguientes variables de entorno:

No se ha definido una variable de entorno. Este error enumera las variables de entorno que no se definieron.

Vea [macros para comandos de compilación personalizada](../../build/reference/common-macros-for-build-commands-and-properties.md) para obtener información sobre macros en el proceso de compilación.
