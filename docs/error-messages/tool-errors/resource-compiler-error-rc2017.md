# <a name="resource-compiler-error-rc2017"></a>Error del compilador de recursos RC2017

secuencia de escape no válida

Una secuencia de escape aparecía cuando no se esperaba una.

Una secuencia de escape, una barra diagonal inversa ( **\\** ) seguida de un número o una letra, solo puede aparecer en un carácter o una constante de cadena.
