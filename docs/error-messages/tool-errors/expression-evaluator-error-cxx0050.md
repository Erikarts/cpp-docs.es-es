# <a name="expression-evaluator-error-cxx0050"></a>Error del evaluador de expresiones CXX0050

lado izquierdo de:: debe ser Class/struct/union

El símbolo del lado izquierdo del operador de resolución de ámbito (`::`) no era una clase, estructura o Unión.

Este error es idéntico a CAN0050.
