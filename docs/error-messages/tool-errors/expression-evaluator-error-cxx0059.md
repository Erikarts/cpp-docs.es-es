# <a name="expression-evaluator-error-cxx0059"></a>Error del evaluador de expresiones CXX0059

el operando izquierdo es una clase que no es un nombre de función

El operando izquierdo de una llamada de función era un nombre de clase y no se pudo resolver en una llamada de función. Omitir el nombre de una función miembro en una expresión puede producir este error.

Este error es idéntico a CAN0059.
