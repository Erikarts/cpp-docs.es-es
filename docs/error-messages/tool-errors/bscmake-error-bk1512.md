# <a name="bscmake-error-bk1512"></a>Error de BSCMAKE BK1512

Filename: se superó la capacidad

BSCMAKE no puede compilar un archivo de información de examen porque el número de definiciones, referencias, módulos u otra información supera el límite.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Para corregir mediante las siguientes posibles soluciones

1. Excluya parte de la información con/EM,/es o/EI.

1. Omita la opción/IU.
