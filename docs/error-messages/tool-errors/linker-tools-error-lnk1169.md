# <a name="linker-tools-error-lnk1169"></a>Error de las herramientas del vinculador LNK1169

se encontraron uno o varios símbolos definidos multiplicados

No se pudo realizar la compilación debido a varias definiciones de uno o más símbolos. Este error está precedido por el error [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).

La opción [/Force](../../build/reference/force-force-file-output.md) o **/Force: Multiple** invalida este error.
