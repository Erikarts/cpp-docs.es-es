# <a name="linker-tools-error-lnk1169"></a>Error de las herramientas del vinculador LNK1169

uno o más símbolos definidos de multiplicar

Error de la compilación debido a varias definiciones de uno o más símbolos. Este error está precedido por error [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).

El [/FORCE](../../build/reference/force-force-file-output.md) o **/Force: Multiple** reemplaza este error.
