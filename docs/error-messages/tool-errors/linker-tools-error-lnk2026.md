# <a name="linker-tools-error-lnk2026"></a>Error de las herramientas del vinculador LNK2026

módulo no seguro de una imagen SAFESEH

[/ SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) se ha especificado, pero un módulo no es compatible con la característica de control de excepciones seguro. Si desea utilizar este módulo con **/SAFESEH**, será necesario volver a compilar el módulo.
