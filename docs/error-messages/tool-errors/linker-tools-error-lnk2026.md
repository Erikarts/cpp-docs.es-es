# <a name="linker-tools-error-lnk2026"></a>Error de las herramientas del vinculador LNK2026

módulo no seguro para la imagen SAFESEH

Se especificó [/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) , pero un módulo no era compatible con la característica de control de excepciones seguro. Si desea usar este módulo con **/SAFESEH**, tendrá que volver a compilar el módulo.
