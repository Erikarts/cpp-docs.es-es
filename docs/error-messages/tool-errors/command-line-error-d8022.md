# <a name="command-line-error-d8022"></a>Error de la línea de comandos D8022

no se puede abrir ' messagefile '

El archivo especificado no se encontraba en el directorio actual o en un directorio especificado en la variable de entorno PATH. El archivo de mensaje contiene un breve resumen de las opciones y la sintaxis de línea de comandos del compilador.

Mueva este archivo al directorio actual o a un directorio de la ruta de acceso actual. Si no se encuentra este archivo, ejecute el programa de instalación para copiarlo desde los discos de distribución.
