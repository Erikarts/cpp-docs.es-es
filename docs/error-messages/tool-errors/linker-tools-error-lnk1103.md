# <a name="linker-tools-error-lnk1103"></a>Error de las herramientas del vinculador LNK1103

información de depuración dañada; Recompile (módulo)

Este error puede deberse a que la compilación finalizó antes de que se creara un archivo de objeto válido.

Vuelva a compilar el archivo objeto especificado. Si la recompilación no soluciona el problema, anote las circunstancias del error, intente aislar el problema y cree un caso de prueba reproducible. A continuación, consulte el archivo de ayuda de soporte técnico o la sección de soporte técnico de uno de los manuales.
