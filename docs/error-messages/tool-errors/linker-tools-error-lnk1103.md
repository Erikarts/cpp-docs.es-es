# <a name="linker-tools-error-lnk1103"></a>Error de las herramientas del vinculador LNK1103

información de depuración dañada; Recompile el módulo

Este error puede producirse porque se canceló la compilación antes de que se creó un archivo de objeto válido.

Vuelva a compilar el archivo objeto dado. Si volver a compilar no soluciona el problema, anote las circunstancias del error, intente aislar el problema y crear un caso de prueba reproducible. A continuación, consulte el archivo de Ayuda de soporte técnico o la sección de soporte técnico en uno de los manuales.
