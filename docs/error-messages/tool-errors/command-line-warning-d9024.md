# <a name="command-line-warning-d9024"></a>Advertencia de la línea de comandos D9024

tipo de archivo de código fuente no reconocido ' nombredearchivo '; se supone que el archivo objeto es

No se reconoció la extensión del archivo especificado. Se supuso que el archivo era un archivo objeto y se pasó al enlazador.

Se reconocen las siguientes extensiones:

- . c (archivo de código fuente de C)

- . CXX (C++ archivo de código fuente)

- . cpp (C++ archivo de código fuente)

- . obj (archivo objeto)

- . lib (archivo de biblioteca)

- . def (archivo de definición de módulo)

- . exp (archivo de exportaciones del enlazador, creado por LINK/LIB)
