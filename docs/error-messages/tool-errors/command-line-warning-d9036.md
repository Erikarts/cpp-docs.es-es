# <a name="command-line-warning-d9036"></a>Advertencia de la línea de comandos D9036

> '*opción\_2*' en lugar de '*opción\_1*'

## <a name="remarks"></a>Observaciones

ADVERTENCIA comandos d9036 es una continuación de la [Advertencia de la línea de comandos D9035](../../error-messages/tool-errors/command-line-warning-d9035.md). Si se prefiere una opción del compilador en lugar de la opción en desuso, se muestra aquí. Para obtener vínculos a más información, vea [Opciones del compilador en desuso y quitadas](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options).
