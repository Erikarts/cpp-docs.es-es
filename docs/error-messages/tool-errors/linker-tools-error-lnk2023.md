# <a name="linker-tools-error-lnk2023"></a>Error de las herramientas del vinculador LNK2023

dll no válida o punto de entrada \<dll o punto de entrada >

El enlazador está cargando una versión incorrecta de msobj90. dll. Asegúrese de que Link. exe y msobj90. dll de la ruta de acceso tienen la misma versión.

Es posible que no exista una dependencia de msobj90. dll. La lista de dependencias de msobj90. dll es:

- Msvcr90.dll

- Kernel32.dll

Compruebe si en el equipo hay otras copias de msobj90. dll que pueden no estar actualizadas.
