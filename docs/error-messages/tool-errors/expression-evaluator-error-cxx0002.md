# <a name="expression-evaluator-error-cxx0002"></a>Error del evaluador de expresiones CXX0002

Error al acceder a la memoria del usuario

La expresión intenta hacer referencia a la memoria que no está asignada al programa que se está depurando.

Este error es idéntico a CAN0002.
