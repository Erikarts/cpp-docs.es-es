# <a name="expression-evaluator-error-cxx0060"></a>Error del evaluador de expresiones CXX0060

el registro no está disponible

Una expresión especificó un registro que no se puede usar. Este error puede deberse a un intento de obtener acceso a un registro que no existe en el equipo que ejecuta.

Este error es idéntico a CAN0060.
