# <a name="resource-compiler-fatal-error-rc1101"></a>Error irrecuperable del compilador de recursos RC1101

> ningún nombre de archivo binario recurso especificado

El resultado de cambiar el nombre (/ fo) opción no iba seguida de un nombre de archivo.

Use la siguiente sintaxis para la opción/fo:

> **/Fo RC** *archivoderesultados* <em>archivo_entrada</em>**.rc**
