# <a name="resource-compiler-fatal-error-rc1101"></a>Error irrecuperable del compilador de recursos RC1101

> no se especificó un nombre de archivo binario de recursos

La opción Rename Output (/FO) no iba seguida de un nombre de archivo.

Use la siguiente sintaxis para la opción/FO:

> **RC/FO** *outfilename* <em>infilename</em> **. RC**
