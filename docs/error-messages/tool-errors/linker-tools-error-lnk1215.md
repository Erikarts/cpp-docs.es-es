# <a name="linker-tools-error-lnk1215"></a>Error de las herramientas del vinculador LNK1215

error en la operación de metadatos (HRESULT): error

El enlazador recibió un error del tiempo de ejecución de .NET al intentar realizar una actualización de metadatos a través del tiempo de ejecución de .NET.

`HRESULT` es el valor HRESULT del método de tiempo de ejecución de .NET. `error` es. Texto proporcionado por la red.

Probablemente tenga un enlazador no coincidente y un tiempo de ejecución de .NET. Reinstale visual C++.
