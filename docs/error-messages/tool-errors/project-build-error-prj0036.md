# <a name="project-build-error-prj0036"></a>Error PRJ0036 al compilar el proyecto

La propiedad ' archivos adicionales ' de la herramienta de implementación web contenía una entrada no válida.

La propiedad archivos adicionales de la página de propiedades de implementación web contenía un error, posiblemente debido a un problema de evaluación de macros. Este error también puede significar que la ruta de acceso tiene un formato incorrecto, que contiene caracteres o combinaciones de caracteres que no son válidos en una ruta de acceso de archivo.

Para resolver este error, corrija la macro o corrija la especificación de la ruta de acceso. La ruta de acceso evaluada es una ruta de acceso absoluta del directorio del proyecto.

Este error también puede significar que uno de los archivos a los que se hace referencia no existe.
