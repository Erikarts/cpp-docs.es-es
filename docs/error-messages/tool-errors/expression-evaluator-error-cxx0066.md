# <a name="expression-evaluator-error-cxx0066"></a>Error del evaluador de expresiones CXX0066

miembro estático no presente

No se pudo encontrar o no se definió un miembro estático de una clase. Este error puede deberse a un miembro de clase estática que se ha declarado pero no se ha definido, o a que solo se ha definido y al que se hace referencia en módulos que no contienen información de depuración.

Este error es idéntico a CAN0066.
