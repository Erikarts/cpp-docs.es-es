# <a name="linker-tools-error-lnk1166"></a>Error de las herramientas del vinculador LNK1166

no se puede ajustar el código en offset = offset, va = valor

LINK no pudo rellenar el código según sea necesario.

No se permiten determinadas instrucciones para cruzar los límites de página en algunos procesadores. VÍNCULO intenta agregar paneles para corregir esta situación. En este caso, el vínculo no pudo solucionar el problema.
