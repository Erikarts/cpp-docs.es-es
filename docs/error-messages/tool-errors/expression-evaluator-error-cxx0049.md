# <a name="expression-evaluator-error-cxx0049"></a>Error del evaluador de expresiones CXX0049

OMF obsoleto: vuelva a vincular el programa

El programa usó un OMF anterior (formato de módulo de objeto).

Volver a vincular el programa con la versión actual del enlazador.

Este error es idéntico a CAN0049.
