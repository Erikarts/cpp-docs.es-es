# <a name="resource-compiler-error-rc2175"></a>Error del compilador de recursos RC2175

el nombre del archivo de recursos no tiene el formato 3,00

El recurso especificado usó un formato anterior a la versión 3,00. El archivo de recursos se debe convertir o volver a crear con el formato de la versión 3,00 o posterior.
