# <a name="project-build-error-prj0015"></a>Error PRJ0015 al compilar el proyecto

Falta el dispositivo nulo en el sistema. No se puede iniciar una compilación.

Este error puede deberse a la escasez de recursos del sistema. Cierre algunas aplicaciones o reinicie el.

También puede deberse a que no tiene suficientes privilegios en el dispositivo nulo del equipo.
