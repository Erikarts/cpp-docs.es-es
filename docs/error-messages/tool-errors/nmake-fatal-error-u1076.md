# <a name="nmake-fatal-error-u1076"></a>Error grave de NMAKE U1076

nombre demasiado largo

Una cadena superó uno de los límites siguientes:

- 1024 caracteres para un nombre de macro.

- 256 caracteres para un nombreruta de destino.

- 2048 caracteres para un comando.
