# <a name="compiler-warning-level-1-c4821"></a>Advertencia del compilador (nivel 1) C4821

> No se puede determinar el tipo de codificación Unicode. Guarde el archivo con la firma (BOM).

El compilador no pudo determinar el tipo de codificación de un archivo. Para resolver esta advertencia, guarde el archivo con un marcador de orden de bytes. Consulte [administrar archivos con codificación](/sql/ssms/solution/manage-files-with-encoding) para obtener más información.
