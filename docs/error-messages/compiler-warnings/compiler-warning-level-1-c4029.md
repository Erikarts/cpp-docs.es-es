# <a name="compiler-warning-level-1-c4029"></a>ADVERTENCIA del compilador (nivel 1) C4029

lista de parámetros formales declarada diferente de la definición

Los tipos de parámetros formales en la declaración de función no coinciden con los de la definición de función. El compilador utiliza la lista de parámetros de la definición.
