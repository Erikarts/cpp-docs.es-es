# <a name="compiler-warning-level-4-c4931"></a>Advertencia del compilador (nivel 4) C4931

> damos por hecho que la biblioteca de tipos se compiló para punteros de bits *numéricos*

No se proporcionó información explícita con el atributo **ptrsize** de la directiva [#import](../../preprocessor/hash-import-directive-cpp.md) ; el compilador llegó a que el tamaño del puntero de la biblioteca de tipos es *Number*.

De forma predeterminada, esta advertencia está desactivada. Vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para más información.
