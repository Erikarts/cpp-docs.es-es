# <a name="compiler-warning-level-3-c4543"></a>Advertencia del compilador (nivel 3) C4543

Texto insertado suprimido por el atributo 'no_injected_text'

El [no_injected_text](../../windows/no-injected-text.md) atributo aparece en el código fuente, lo que significa que el compilador impide que los atributos de inyección de código.

Esta advertencia es para recordarle que los atributos no podrá insertar código.
