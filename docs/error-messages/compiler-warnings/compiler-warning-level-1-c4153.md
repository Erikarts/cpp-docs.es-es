# <a name="compiler-warning-level-1-c4153"></a>Advertencia del compilador (nivel 1) C4153

conversión de puntero a función/datos en expresión

Un puntero a función se ha convertido a un puntero a datos o desde este. Esta conversión se permite en las extensiones de Microsoft (/Ze), pero no en ANSI C.
