# <a name="compiler-warning-level-1-c4168"></a>Advertencia del compilador (nivel 1) C4168

límite del compilador: no hay tipos de depurador, elimine la base de datos de programa 'database' y repita la compilación

El archivo de base de datos de programa debe volver a compilarse para dar cabida a todos los tipos del programa.
