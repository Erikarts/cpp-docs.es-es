# <a name="compiler-warning-level-4-c4505"></a>Advertencia del compilador (nivel 4) C4505

' función ': se ha quitado la función local sin referencia

La función especificada es local y no se hace referencia a ella en el cuerpo del módulo; por lo tanto, la función es Código inactivo.

El compilador no generó código para esta función muerta.
