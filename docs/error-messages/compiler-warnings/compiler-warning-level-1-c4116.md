# <a name="compiler-warning-level-1-c4116"></a>ADVERTENCIA del compilador (nivel 1) C4116

definición de tipo sin nombre entre paréntesis

Una estructura, una Unión o un tipo enumerado sin nombre se define en una expresión entre paréntesis. La definición de tipo no tiene sentido.

En una llamada de función de C, la definición tiene ámbito global. En una C++ llamada de función, la definición tiene el mismo ámbito que la función a la que se llama.
