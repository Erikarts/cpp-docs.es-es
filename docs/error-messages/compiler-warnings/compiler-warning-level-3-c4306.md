# <a name="compiler-warning-level-3-c4306"></a>Advertencia del compilador (nivel 3) C4306

> '*Identifier*': conversión de '*tipo1*' a '*tipo2*' de mayor tamaño

El identificador es un tipo convertido en un puntero mayor. Los bits altos sin rellenar del nuevo tipo tendrán relleno cero.

Esta advertencia puede indicar una conversión no deseada. Es posible que el puntero resultante no sea válido.
