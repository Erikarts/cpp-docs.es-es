# <a name="compiler-warning-level-1-c4541"></a>Advertencia del compilador (nivel 1) C4541

' Identifier ' usado en el tipo polimórfico ' type ' con/GR-; puede producirse un comportamiento impredecible

Ha intentado usar una característica que requiere información de tipo en tiempo de ejecución sin habilitar la información de tipo en tiempo de ejecución. Vuelva a compilar con [/gr](../../build/reference/gr-enable-run-time-type-information.md).
