# <a name="compiler-warning-level-1-c4022"></a>ADVERTENCIA del compilador (nivel 1) C4022

' función ': no coincide el puntero para el parámetro real ' número '

El tipo de puntero del parámetro real difiere del tipo de puntero del parámetro formal correspondiente. El parámetro real se pasa sin cambiar.
