# <a name="compiler-warning-level-4-c4513"></a>Advertencia del compilador (nivel 4) C4513

' Class ': no se pudo generar el destructor

El compilador no puede generar un destructor predeterminado para la clase especificada; no se creó ningún destructor. El destructor está en una clase base a la que no se puede tener acceso a la clase derivada. Si la clase base tiene un destructor privado, conviértalo en público o protegido.
