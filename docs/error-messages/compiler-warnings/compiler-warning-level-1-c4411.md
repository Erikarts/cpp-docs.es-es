# <a name="compiler-warning-level-1-c4411"></a>Advertencia del compilador (nivel 1) C4411

' Identifier ': el símbolo se resuelve como un registro de desplazamiento

El identificador es un símbolo local que se resuelve como un registro de desplazamiento y, por lo tanto, se puede usar en un operando con otro símbolo.
