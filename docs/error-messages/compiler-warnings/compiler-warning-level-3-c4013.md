# <a name="compiler-warning-level-3-c4013"></a>ADVERTENCIA del compilador (nivel 3) C4013

' función ' sin definir; se supone que extern devuelve int

El compilador encontró una llamada a una función sin definir.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Posibles causas del error:

1. Ortografía incorrecta del nombre de la función

1. Funciones externas no prototipos como `extern`
