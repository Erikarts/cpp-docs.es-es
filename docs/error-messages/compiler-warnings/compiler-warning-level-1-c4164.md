# <a name="compiler-warning-level-1-c4164"></a>Advertencia del compilador (nivel 1) C4164

'identifier': función intrínseca no declarada

La función intrínseca especificada no se ha declarado; puede que necesite #include el archivo de encabezado correspondiente.
