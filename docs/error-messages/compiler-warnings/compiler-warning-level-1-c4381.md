# <a name="compiler-warning-level-1-c4381"></a>Advertencia del compilador (nivel 1) C4381

' function1 ': el método no público ' función2 ' no implementará el método de interfaz

Una clase debe implementar toda la función en una interfaz. Una clase puede satisfacer esta condición si una de sus clases base implementa la función. Sin embargo, la función debe implementarse como una función pública.
