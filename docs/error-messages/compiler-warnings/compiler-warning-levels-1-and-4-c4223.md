# <a name="compiler-warning-levels-1-and-4-c4223"></a>Advertencia del compilador (nivels 1 and 4) C4223

ha utilizado una extensión no estándar: matriz de un valor que no sea l convertida a puntero

En C estándar, no se puede convertir una matriz que no sea lvalue a un puntero. Con las extensiones de Microsoft ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)), es posible.
