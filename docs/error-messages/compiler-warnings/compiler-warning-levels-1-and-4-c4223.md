# <a name="compiler-warning-levels-1-and-4-c4223"></a>Advertencia del compilador (niveles 1 y 4) C4223

se ha utilizado una extensión no estándar: matriz no de valor l convertida en puntero

En C estándar, no se puede convertir una matriz que no sea de valor l en un puntero. Con las extensiones predeterminadas de Microsoft ([/ze](../../build/reference/za-ze-disable-language-extensions.md)), puede.
