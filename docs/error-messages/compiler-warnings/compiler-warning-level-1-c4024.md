# <a name="compiler-warning-level-1-c4024"></a>Compilador advertencia (nivel 1) C4024

'function': tipos distintos para el parámetro formal y real 'número'

Los parámetros formales y reales correspondientes tienen distintos tipos. El compilador pasa el parámetro real sin cambios. La función de recepción convierte el tipo de parámetro para el tipo esperado.
