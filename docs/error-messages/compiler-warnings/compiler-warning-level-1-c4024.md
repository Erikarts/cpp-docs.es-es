# <a name="compiler-warning-level-1-c4024"></a>ADVERTENCIA del compilador (nivel 1) C4024

' función ': tipos diferentes para el parámetro formal y real ' número '

Los parámetros formales y reales correspondientes tienen tipos diferentes. El compilador pasa el parámetro real sin cambiar. La función receptora convierte el tipo de parámetro en el tipo esperado.
