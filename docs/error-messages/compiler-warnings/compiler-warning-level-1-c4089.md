# <a name="compiler-warning-level-1-c4089"></a>Compilador advertencia (nivel 1) C4089

'function': tipos distintos en parámetro real 'número', parámetro formal 'número'

Los parámetros formales y reales correspondientes tienen tipos diferentes. El parámetro real se pasa sin cambios. La función convierte el parámetro real al tipo especificado en la definición de función.
