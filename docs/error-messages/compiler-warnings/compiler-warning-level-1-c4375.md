# <a name="compiler-warning-level-1-c4375"></a>Advertencia del compilador (nivel 1) C4375

el método no público ' method2 ' no invalida ' method1 '

Un tipo que implementa otro tipo definido por un método de invalidación, pero el reemplazo no era público. Por consiguiente, el método no invalida el método de tipo base.
