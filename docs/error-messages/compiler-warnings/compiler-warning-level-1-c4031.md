# <a name="compiler-warning-level-1-c4031"></a>ADVERTENCIA del compilador (nivel 1) C4031

la segunda lista de parámetros formales es más larga que la primera

Una función se declara con diferentes parámetros formales. El compilador usa los parámetros formales especificados en la primera declaración.
