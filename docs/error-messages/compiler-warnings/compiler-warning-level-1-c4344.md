# <a name="compiler-warning-level-1-c4344"></a>Advertencia del compilador (nivel 1) C4344

cambio de comportamiento: el uso de argumentos de plantilla explícitos da como resultado una llamada a ' función '

Una llamada a una función que usa argumentos de plantilla explícitos llama a una función diferente que si no se hubieran especificado argumentos explícitos
