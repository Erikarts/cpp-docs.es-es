# <a name="compiler-warning-level-1-c4379"></a>Advertencia del compilador (nivel 1) C4379

Este compilador no admite la versión de la versión de Common Language Runtime. El uso de esta versión puede producir resultados inesperados.

Tiene una versión anterior del Common Language Runtime en el equipo, pero no la versión actual. Para resolver C4379, instale la versión de la Common Language Runtime que se incluye con el compilador.
