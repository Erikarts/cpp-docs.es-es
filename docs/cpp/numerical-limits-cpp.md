# <a name="numerical-limits-c"></a>Límites numéricos (C++)

Los dos archivos de inclusión estándar, \<limits. h > y \<float. h >, definen los límites numéricos, o los valores máximo y mínimo que puede contener una variable de un tipo determinado. Se garantiza que estos valores mínimo y máximo son portátiles para cualquier C++ compilador que use la misma representación de datos que ANSI C. El archivo de inclusión \<limits. h > define los [límites numéricos de los tipos enteros](../cpp/integer-limits.md)y \<float. h > define los [límites numéricos de los tipos flotantes](../cpp/floating-limits.md).
