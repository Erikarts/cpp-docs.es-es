# <a name="work-with-odbc-and-drivers"></a>Trabajar con ODBC y controladores

ODBC permite escribir aplicaciones que no dependen de un solo sistema de administración de bases de datos (DBMS). Puede escribir una aplicación que pueda funcionar en varios DBMS para los que los usuarios tengan el controlador ODBC específico de DBMS adecuado.
