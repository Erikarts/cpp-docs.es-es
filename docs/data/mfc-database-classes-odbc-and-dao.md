# <a name="mfc-database-classes"></a>Clases de bases de datos MFC

> [!NOTE]
> Microsoft recomienda usar ODBC a través del controlador ODBC 13 de Microsoft (o posterior) para SQL Server para nuevos proyectos. DAO ya no es compatible.

Para obtener más información, vea los temas siguientes:

- [Historial de controladores para Microsoft SQL Server](/sql/connect/connect-history)
