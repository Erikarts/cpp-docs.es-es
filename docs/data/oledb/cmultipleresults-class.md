# <a name="cmultipleresults-class"></a>CMultipleResults (Clase)

Si desea que un comando controle varios conjuntos de resultados, utilice `CMultipleResults` para el argumento de plantilla de `CCommand` *TMultiple*.

## <a name="syntax"></a>Sintaxis

```cpp
class CMultipleResults
```

## <a name="remarks"></a>Observaciones

Para controlar varios conjuntos de resultados, [CCommand](../../data/oledb/ccommand-class.md) debe heredar de esta clase.

## <a name="requirements"></a>Requisitos

**Encabezado:** atldbcli
