# <a name="activex-controls"></a>Controles ActiveX

En Visual C++ puede crear controles ActiveX con MFC o ATL.

>[!IMPORTANT]
> ActiveX es una tecnología heredada que no se recomienda para nuevo desarrollo. Muchas de las funcionalidades de los controles ActiveX se pueden realizar en una forma mucho más sencilla y segura con tecnologías modernas como HTML5 y JavaScript, las extensiones del explorador moderno o WebAssembly módulos. Para obtener más información, consulte [un salto desde el pasado, 2ª parte: Adiós a ActiveX, VBScript, attachEvent...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) y [mensajería nativo](/microsoft-edge/extensions/guides/native-messaging) y [las extensiones de Microsoft Edge](/microsoft-edge/extensions) y [WebAssembly](https://webassembly.org/).
