# <a name="tn036-using-cformview-with-appwizard-and-classwizard"></a>TN036: Usar CFormView con AppWizard y ClassWizard

Esta nota técnica describe cómo modificar una aplicación mediante AppWizard generado para que usa un `CFormView` en lugar del predeterminado `CView` como su clase de vista principal. Esto es compatible directamente con esta versión de Visual C++.
