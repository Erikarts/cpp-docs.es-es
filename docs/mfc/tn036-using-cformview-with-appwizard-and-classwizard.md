# <a name="tn036-using-cformview-with-appwizard-and-classwizard"></a>TN036: Usar CFormView con AppWizard y ClassWizard

En esta nota técnica se describe cómo modificar una aplicación generada por el Asistente para aplicaciones para que use un `CFormView` en lugar del `CView` predeterminado como su clase de vista principal. Esto se admite directamente con esta versión de Visual C++.
