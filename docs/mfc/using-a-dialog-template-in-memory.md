# <a name="using-a-dialog-template-in-memory"></a>Usar una plantilla de cuadro de diálogo en memoria

En lugar de usar los métodos proporcionados en la [tabla de creación de cuadros de diálogo](../mfc/creating-a-dialog-class-with-code-wizards.md), puede crear cualquier tipo de cuadro de diálogo indirectamente a partir de una plantilla de cuadro de diálogo en la memoria. Para obtener más información, vea la clase [CDialog](../mfc/reference/cdialog-class.md) en la *referencia de MFC*.
