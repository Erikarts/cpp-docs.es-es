# <a name="making-owner-drawn-header-controls"></a>Crear controles de encabezado dibujados por el propietario

Puede definir elementos individuales de un control de encabezado ([CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)) para que sean elementos dibujados por el propietario. Para obtener más información, vea [controles de encabezado dibujados](/windows/win32/Controls/header-controls) por el propietario en el Windows SDK.
