# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>Desplazar, organizar, ordenar y buscar en los controles de lista

De forma predeterminada, los controles de lista ([CListCtrl](../mfc/reference/clistctrl-class.md)) son desplazables. Para obtener más información, vea [posición de desplazamiento](/windows/win32/Controls/using-list-view-controls) en el Windows SDK y la función miembro de [desplazamiento](../mfc/reference/clistctrl-class.md#scroll) .

Puede llamar `CListCtrl` a funciones miembro para organizar elementos de lista en el control, ordenar elementos y buscar elementos concretos. Para obtener más información, vea [usar controles ListView](/windows/win32/Controls/using-list-view-controls) en el Windows SDK y los miembros [CListCtrl](../mfc/reference/clistctrl-class.md) [Arrange](../mfc/reference/clistctrl-class.md#arrange), [SortItems](../mfc/reference/clistctrl-class.md#sortitems)y [FindItem](../mfc/reference/clistctrl-class.md#finditem).
