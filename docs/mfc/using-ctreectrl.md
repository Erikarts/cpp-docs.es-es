# <a name="using-ctreectrl"></a>Usar CTreeCtrl

Un control de árbol, representado por la clase [CTreeCtrl](../mfc/reference/ctreectrl-class.md), es una ventana que muestra una lista jerárquica de elementos, como los encabezados en un documento, las entradas de un índice, o los archivos y directorios en un disco. Cada elemento se compone de una etiqueta y una imagen de mapa de bits opcional, y cada elemento puede tener una lista de subelementos asociados con él. Al hacer clic en un elemento, el usuario puede expandir y contraer la lista asociada de subelementos. El árbol de directorios en el panel izquierdo del explorador de archivos es un ejemplo de un control de árbol.
