# <a name="internal-classes"></a>Clases internas

Las siguientes clases se usan internamente en MFC. Por integridad, en esta sección se describen estas clases internas, pero no están destinadas a usarse directamente en el código.

## <a name="in-this-section"></a>En esta sección

|Clase|Descripción|
|-----------|-----------------|
|`CCommandManager Class`|La `CCommandManager` clase administra los comandos y su asociación con imágenes.|
|`CDialogImpl Class (MFC)`|Proporciona los detalles de implementación para las clases de basado en el cuadro de diálogo, como `CDialogEx`.|
|`CDocumentAdapter Class`|Implementa la interfaz IDocument necesaria para buscar y organizar los controladores.|
|`CFrameImpl Class`|La `CFrameImpl` clase controla la personalización de la barra de herramientas para las siguientes clases de ventana de marco: [Clase CFrameWndEx](../../mfc/reference/cframewndex-class.md), [CMDIFrameWndEx (clase)](../../mfc/reference/cmdiframewndex-class.md), y [COleIPFrameWndEx (clase)](../../mfc/reference/coleipframewndex-class.md).|
|`CFullScreenImpl Class`|Implementa funciones de pantalla completa comunes a las aplicaciones SDI y MDI.|
|`CMDIClientAreaWnd Class`|La `CMDIClientAreaWnd` clase es una clase auxiliar que simplifica la implementación de las pestañas MDI y grupos con pestañas de MDI.|
|`CMemDC Class`|Una clase auxiliar para un contexto de dispositivo de memoria. El contexto de dispositivo de memoria admite el dibujo fuera de la pantalla.|
|`CMenuHash Class`|Lee el estado de `CMFCToolBar` objetos desde un archivo o escribe el estado en un archivo.|
|`CMenuImages Class`|Proporciona la funcionalidad para mostrar imágenes predefinidas, como botones para cerrar, maximizar los botones, botones de radio y flechas.|
|`CMFCCaptionButtonEx Class`|Contiene el área no cliente de los botones de título del sistema.|
|`CMFCCaptionMenuButton Class`|Representa un botón de menú que se encuentra en el área de título de un panel de tareas o una barra de herramientas.|
|`CMFCColorPropertySheet Class`|Utilizado por [CMFCColorDialog (clase)](../../mfc/reference/cmfccolordialog-class.md) para proporcionar un cuadro de diálogo con pestañas.|
|`CMFCControlBarImpl Class`|Proporciona los detalles de implementación de control basado en la barra de clases, como `CMFCReBar` y `CMFCToolBar`.|
|`CMFCControlRenderer Class`|Una clase auxiliar que controla la representación de imágenes.|
|`CMFCControlRendererInfo Class`|Una clase auxiliar para pasar parámetros a `CMFCControlRenderer` clase.|
|`CMFCCustomizeButton Class`|Admite un comando de menú o un botón de barra de herramientas que se inicia un cuadro de diálogo de personalización.|
|`CMFCCustomizeMenuButton Class`|Representa un solo botón en un menú de personalización.|
|`CMFCDropDownListBox Class`|Proporciona la funcionalidad de cuadro de lista desplegable para los cuadros combinados de cinta de opciones.|
|`CMFCMousePropertyPage Class`|Página de propiedades que permite al usuario personalizar el comportamiento del mouse.|
|`CMFCOutlookBarPaneAdapter Class`||
|`CMFCOutlookBarPaneButton Class`|Implementa un botón que agrega el marco de trabajo a un `CMFCOutlookBarPane` objeto.|
|`CMFCOutlookBarPaneList Class`|Extiende la `CMFCOutlookBarPane` clase para proporcionar el estilo visual de Outlook.|
|`CMFCOutlookBarScrollButton Class`|Proporciona funcionalidad de desplazamiento a la `CMFCOutlookBarTabCtrl` clase.|
|`CMFCOutlookBarToolBar Class`|Implementa una barra de herramientas que tiene el estilo visual de Outlook 2003.|
|`CMFCPropertySheetCategoryInfo Class`|Representa un nodo en un control de árbol.|
|`CMFCPropertySheetTabCtrl Class`|Extiende la `CMFCTabCtrl` clase para proporcionar funcionalidad de hoja de propiedades.|
|`CMFCProperySheetListBox Class`|Extiende la `CListBox` clase para proporcionar funcionalidad de hoja de propiedades.|
|`CMFCReBarState Class`||
|`CMFCRibbonCaptionButton Class`|Implementa un botón de título del sistema.|
|`CMFCRibbonCmdUI Class`|Las actualizaciones de la cinta de opciones de la barra objetos según sea necesario cuando la aplicación está inactiva.|
|`CMFCRibbonCommandsListBox Class`||
|`CMFCRibbonDefaultPanelButton Class`||
|`CMFCRibbonGalleryIcon Class`|Representa un elemento en un `CMFCRibbonGallery` objeto.|
|`CMFCRibbonKeyboardCustomizeDialog Class`|Implementa un cuadro de diálogo de personalización de teclado para las aplicaciones que contienen barras de cinta de opciones.|
|`CMFCRibbonKeyTip Class`|Implementa una ventana emergente que muestra una sugerencia de clave para una barra de cinta.|
|`CMFCRibbonPanelMenu Class`||
|`CMFCRibbonPanelMenuBar Class`|Representa una barra de menús emergentes en la barra de cinta de opciones.|
|`CMFCRibbonQuickAccessToolBar Class`|Implementa la barra de herramientas de acceso rápido para un elemento de la cinta de opciones.|
|`CMFCRibbonRichEditCtrl Class`|Implementa un control de edición que se encuentra en una barra de cinta.|
|`CMFCRibbonTab Class`|Representar una tabulación en una barra de cinta.|
|`CMFCShadowRenderer Class`||
|`CMFCShowAllButton Class`|Representa un botón que se encuentra en la parte inferior de un menú emergente que se expande para mostrar los comandos ocultos.|
|`CMFCStatusBarPaneInfo Class`|Describe el contenido y la apariencia de un `CMFCStatusBar` panel.|
|`CMFCTabButton Class`|Proporciona la funcionalidad de botón de pestaña a los controles de ficha.|
|`CMFCTabInfo Class`|Proporciona información sobre los controles de pestaña.|
|`CMFCTasksPaneFrameWnd Class`|Administra los botones de título personalizado de un panel de tareas.|
|`CMFCTasksPanePropertyPage Class`|Administra la relación entre un `CMFCTasksPane` objeto y su nombre.|

|`CMFCToolBarButtonsListButton Class`|Muestra una lista de imágenes que el usuario puede seleccionar durante la personalización en el **apariencia del botón** cuadro de diálogo.|
|`CMFCToolBarDateTimeCtrlImpl Class`|Implementa un origen de colocación para `CMFCToolBar` objetos.|
|`CMFCToolBarDropSource Class`|Implementa un origen de colocación para `CMFCToolBar` objetos.|
|`CMFCToolBarDropTarget Class`|Implementa un destino de colocación para `CMFCToolBar` objetos.|
|`CMFCToolBarEditCtrl Class`|Proporciona funcionalidad de control de edición para `CMFCToolBarEditBoxButton` objetos.|

|`CMFCToolBarNameDialog Class`|Representa un cuadro de diálogo que permite al usuario especificar un nuevo título de la barra de herramientas.|
|`CMFCToolBarsCommandsListBox Class`|Representa un cuadro de lista que contiene una lista de comandos de la barra de herramientas.|
