# <a name="cmdichildwndex-class"></a>CMDIChildWndEx (clase)

La `CMDIChildWndEx` clase proporciona la funcionalidad de una ventana secundaria de interfaz de varios documentos (MDI) de Windows. Amplía la funcionalidad de [CMDIChildWnd (clase).](../../mfc/reference/cmdichildwnd-class.md) El marco requiere esta clase cuando una aplicación MDI utiliza determinadas clases MFC.

Para obtener más información, vea el código fuente ubicado en la carpeta **VC\\atlmfc\\src\\mfc** de la instalación de Visual Studio.

## <a name="syntax"></a>Sintaxis

```
class CMDIChildWndEx : public CMDIChildWnd
```

```
    CPane* pBar,

```

*pBar*<br/>

Llame a este método cuando desee acoplar varios paneles en orden predefinido.

`GetDocumentName`es llamado por el marco de trabajo cuando guarda el estado de todos los documentos abiertos. La cadena devuelta se escribe en el registro.

Debe realizar las acciones anteriores solo si tiene la intención de guardar y restaurar documentos.
