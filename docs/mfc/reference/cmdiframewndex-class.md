# <a name="cmdiframewndex-class"></a>CMDIFrameWndEx (clase)

## <a name="syntax"></a>Sintaxis

```
class CMDIFrameWndEx : public CMDIFrameWnd
```

##  <a name="addpane"></a>Cmdiframewndex (:: AddPane

Registra un panel con el administrador de acoplamiento.

```
BOOL AddPane(
    CBasePane* pControlBar,
    BOOL bTail=TRUE);
```

### <a name="parameters"></a>Parámetros

*pControlBar*<br/>
de Puntero al panel que se va a registrar.

*bTail*<br/>
de Especifica si se va a agregar este panel al final de la lista.

- BCGP_MDI_CREATE_VERT_GROUP: puede crear un grupo de pestañas vertical.

- BCGP_MDI_CREATE_HORZ_GROUP: puede crear un grupo de pestañas horizontal.

- BCGP_MDI_CAN_MOVE_PREV: puede desplace una pestaña al grupo de pestañas anterior.

- BCGP_MDI_CAN_MOVE_NEXT: puede desplace una pestaña al siguiente grupo de pestañas.

### <a name="remarks"></a>Observaciones

Cuando está habilitada la característica de grupos con pestañas MDI, debe saber qué operaciones se permiten en las pestañas de una ventana determinada. Este método analiza el diseño actual de las ventanas con pestañas y devuelve una combinación de marcas que se pueden usar para compilar, por ejemplo, un menú contextual.

Puede crear un nuevo grupo de pestañas vertical cuando todas las ventanas con pestañas estén alineadas verticalmente o cuando solo haya una ventana con pestañas.

Puede crear un nuevo grupo de pestañas horizontal cuando todas las ventanas con pestañas estén alineadas horizontalmente o cuando solo haya una ventana con pestañas.

Puede desplace una pestaña al grupo anterior solo si hay más de una pestaña en una ventana con pestañas.

Puede desplace una pestaña al siguiente grupo solo si hay más de una pestaña en una ventana con pestañas.

##  <a name="savemdistate"></a>Cmdiframewndex (:: SaveMDIState


### <a name="parameters"></a>Parámetros

*lpszProfileName*<br/>

### <a name="return-value"></a>Valor devuelto

La cadena devuelta se guardará en el registro como un identificador de documento. Para obtener más información, vea [cmdichildwndex (:: GetDocumentName](../../mfc/reference/cmdichildwndex-class.md#getdocumentname).
