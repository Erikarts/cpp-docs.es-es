# <a name="cmfctoolbarimages-class"></a>Clase CMFCToolBarImages

Imágenes de una barra de herramientas. La clase `CMFCToolBarImages` administra las imágenes de la barra de herramientas cargadas de los recursos de la aplicación o de los archivos.
Para obtener más información, consulte el código fuente que se encuentra en la **\\VC\\src\\MFC** de la instalación de Visual Studio.
## <a name="syntax"></a>Sintaxis

```
class CMFCToolBarImages : public CObject
```

##  <a name="draw"></a>CMFCToolBarImages::D RAW

Dibuja una sola imagen de la barra de herramientas.

```
BOOL Draw(
    CDC* pDC,
    int x,
    int y,
    int iImageIndex,
    BOOL bHilite=FALSE,
    BOOL bDisabled=FALSE,
    BOOL bIndeterminate=FALSE,
    BOOL bShadow=FALSE,
    BOOL bInactive=FALSE,
    BYTE alphaSrc=255);
```

### <a name="parameters"></a>Parámetros

*pDC*<br/>
de Un puntero a un contexto de dispositivo.

*x*<br/>
de Coordenada X del lado izquierdo del rectángulo donde se va a dibujar la imagen.

*y*<br/>
de Coordenada Y de la parte superior del rectángulo donde se va a dibujar la imagen.

*iImageIndex*<br/>
de Índice de base cero de la imagen que se va a mostrar.

*bHilite*<br/>
de TRUE si la imagen se va a resaltar; en caso contrario, FALSE.

*bDisabled*<br/>
de TRUE si la imagen se va a dibujar en el estilo deshabilitado; en caso contrario, FALSE.

*bIndeterminate*<br/>
de TRUE si la imagen se va a dibujar en el estilo de estado indeterminado; en caso contrario, FALSE.

*bShadow*<br/>
de TRUE si la imagen se va a dibujar con una sombra paralela; en caso contrario, FALSE.

*bInactive*<br/>
de TRUE si la imagen se va a dibujar en el estilo de estado inactivo; en caso contrario, FALSE.

*alphaSrc*<br/>
de Valor del canal alfa (opacidad). Un valor de 255 significa que la imagen se dibuja opaca. Un valor de 0 significa que la imagen se dibuja transparente. Este valor solo se usa para imágenes de color de 32 bits y para imágenes que muestran un estilo de cristal de Windows Vista.

### <a name="return-value"></a>Valor devuelto

##  <a name="grayimages"></a>CMFCToolBarImages::GrayImages

Atenúa las imágenes de la barra de herramientas para que parezcan deshabilitadas.

```

```

### <a name="parameters"></a>Parámetros

*nGrayImageLuminancePercentage*<br/>
de Porcentaje de luminancia.

### <a name="return-value"></a>Valor devuelto

Para ello, puede llamar a [CMFCToolBarImages:: Load](#load) o [CMFCToolBarImages:: UpdateImage](#updateimage) (solo para las imágenes definidas por el usuario) o al llamar a [CMFCToolBarImages:: Clear](#clear) y volver a agregar las imágenes llamando a [CMFCToolBarImages:: AddIcon](#addicon) o [CMFCToolBarImages:: AddImage](#addimage).

```
BOOL UpdateImage(
    int iImage,
    HBITMAP hbmp);
```

### <a name="parameters"></a>Parámetros

*iImage*<br/>
de Índice de base cero de la imagen que se va a actualizar.

*hbmp*<br/>
de Identificador del mapa de bits del que se va a actualizar la imagen.

### <a name="return-value"></a>Valor devuelto

TRUE si la imagen se actualizó correctamente; FALSE si la lista de imágenes no está definida por el usuario o es temporal.
