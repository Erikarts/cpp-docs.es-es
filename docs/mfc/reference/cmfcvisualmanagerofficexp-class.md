# <a name="cmfcvisualmanagerofficexp-class"></a>CMFCVisualManagerOfficeXP (clase)

El `CMFCVisualManagerOfficeXP` proporciona a una aplicación un aspecto de Microsoft Office XP.

## <a name="syntax"></a>Sintaxis

```
class CMFCVisualManagerOfficeXP : public CMFCVisualManager
```

## <a name="members"></a>Miembros

Todos los miembros de esta clase son funciones virtuales que se derivan el antecesor de esta clase, [CMFCVisualManager (clase)](../../mfc/reference/cmfcvisualmanager-class.md).

## <a name="remarks"></a>Comentarios

Usa el `CMFCVisualManagerOfficeXP` clase para cambiar la apariencia visual de la aplicación para que se parecen a las de Microsoft Office XP.
