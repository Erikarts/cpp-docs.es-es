## <a name="remarks"></a>Observaciones

Un botón de galería se comporta igual que un botón de menú normal, excepto que muestra una galería cuando un usuario la abre. Al seleccionar un elemento en una galería, el marco de trabajo envía el mensaje WM_COMMAND junto con el identificador de comando del botón. Al controlar el mensaje, debe llamar a [CMFCRibbonGallery::GetLastSelectedItem](#getlastselecteditem) para determinar qué elemento se seleccionó de la galería.

## <a name="cmfcribbongalleryaddgroup"></a><a name="addgroup"></a>CMFCRibbonGallery::AddGroup

Agrega un nuevo grupo a la galería.

```cpp
void AddGroup(
    LPCTSTR lpszGroupName,
    UINT uiImagesPaletteResID,
    int cxPaletteImage);

void AddGroup(
    LPCTSTR lpszGroupName,
    CMFCToolBarImages& imagesGroup);

void AddGroup(
    LPCTSTR lpszGroupName,
    int nIconsNum);
```

### <a name="parameters"></a>Parámetros

*lpszGroupName*<br/>
[en] Especifica el nombre del grupo.

*uiImagesPaletteResID*<br/>
[en] Especifica el identificador de recurso de la lista de imágenes que contiene las imágenes del grupo.

*cxPaletteImage*<br/>
[en] Especifica el ancho en píxeles de una imagen.

*imagesGroup*<br/>
[en] Una referencia a la lista de imágenes que contiene imágenes de grupo.

*nIconsNum*<br/>
[en] Especifica el número de iconos del grupo. Este parámetro debe especificarse solo para grupos personalizados (dibujados por el propietario).

### <a name="remarks"></a>Observaciones

Puede dividir los elementos de una galería de la cinta de opciones en varios grupos llamando a este método. Cada grupo puede tener un título.

## <a name="cmfcribbongalleryaddsubitem"></a><a name="addsubitem"></a>CMFCRibbonGallery::AddSubItem

Agrega un nuevo elemento de menú al menú desplegable.

```cpp
void AddSubItem(
    CMFCRibbonBaseElement* pSubItem,
    int nIndex=-1,
    BOOL bOnTop=FALSE);
```

### <a name="parameters"></a>Parámetros

*pSubItem*<br/>
[en] Un puntero al elemento que se va a agregar al menú.

*nIndex*<br/>
[en] Especifica el índice de base cero de una ubicación donde insertar el elemento.

*bOnTop*<br/>
[en] TRUE para especificar que el elemento debe insertarse antes de la galería de la cinta de opciones; de lo contrario, FALSE.

### <a name="remarks"></a>Observaciones

Puede combinar galerías emergentes con elementos de menú emergente llamando a este método. Los elementos del menú se pueden colocar antes o después de la galería.

Para insertar el elemento antes de la galería, establezca *bOnTop en* TRUE. Establezca *bOnTop en* FALSE para insertar el elemento debajo de la galería.

> [!NOTE]
> El parámetro *nIndex* especifica el índice de inserción tanto en la parte superior de la galería como en la parte inferior de la galería. Por ejemplo, si necesita insertar un elemento una posición antes de la galería, establezca *nIndex* en 1 y *bOnTop en* TRUE. Del mismo modo, si necesita insertar un elemento una posición debajo de la galería, establezca *nIndex* en 1 y *bOnTop en* FALSE.

*nIconsNum*<br/>

## <a name="cmfcribbongalleryenablemenuresize"></a><a name="enablemenuresize"></a>CMFCRibbonGallery::EnableMenuResize

```cpp

    BOOL bEnable = TRUE,
    BOOL bVertcalOnly = FALSE);
```

### <a name="parameters"></a>Parámetros

*bHabilitar*<br/>
[en] TRUE para habilitar el cambio de tamaño del menú; de lo contrario, FALSE.
