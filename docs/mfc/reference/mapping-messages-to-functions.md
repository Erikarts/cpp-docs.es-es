# <a name="mapping-messages-to-functions"></a>Asignar mensajes a funciones

El [Asistente para clases](mfc-class-wizard.md) le permite enlazar controladores de mensajes (funciones miembro de clases de la interfaz de usuario de MFC) con los mensajes generados por los recursos de la aplicación. Utilizan [mapas de mensajes de MFC](../../mfc/messages-and-commands-in-the-framework.md) para crear el enlace.

Cuando se usa Vista de clases para crear una nueva clase derivada de una de las clases de .NET Framework, coloca automáticamente una clase completa y funcional en los archivos de encabezado (. h) y de implementación (. cpp) que se especifiquen.

> [!NOTE]
>  Para agregar una nueva clase que no controle los mensajes, cree la clase directamente en el editor de texto.

### <a name="to-define-or-remove-a-message-handler-using-the-class-wizard"></a>Para definir o quitar un controlador de mensajes mediante el Asistente para clases

1. En **vista de clases**, haga clic con el botón secundario en la clase.

1. En el menú contextual, elija [Asistente para clases](mfc-class-wizard.md).
