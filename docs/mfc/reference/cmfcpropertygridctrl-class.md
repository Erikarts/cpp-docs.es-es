# <a name="cmfcpropertygridctrl-class"></a>Clase Cmfcpropertygridctrl (

Para obtener más información, consulte el código fuente ubicado en la carpeta **VC\\atlmfc\\src\\MFC** de la instalación de Visual Studio.

Admite un control de cuadrícula de propiedades editables que puede mostrar propiedades en orden alfabético o jerárquico.

## <a name="syntax"></a>Sintaxis

```
class CMFCPropertyGridCtrl : public CWnd
```

## <a name="remarks"></a>Comentarios

La `CMFCPropertyGridCtrl` clase muestra un control de cuadrícula de propiedades que contiene las propiedades editables derivadas de la clase [CMFCPropertyGridProperty](../../mfc/reference/cmfcpropertygridproperty-class.md) . Cada propiedad puede representar un tipo y puede contener subelementos. El control de cuadrícula de propiedades admite un área de tamaño variable en la parte inferior que puede mostrar la descripción de una propiedad seleccionada.

Para usar un control de cuadrícula de propiedades, `CMFCPropertyGridCtrl` construya un objeto y, a continuación, llame al método [cmfcpropertygridctrl (:: Create](#create) . Use el método [cmfcpropertygridctrl (:: AddProperty](#addproperty) para agregar propiedades a la lista.

## <a name="selection-properties"></a>Propiedades de selección

En lugar de representar un valor, un elemento de propiedad puede iniciar un cuadro de diálogo que permite al usuario seleccionar un color, un archivo o una fuente.

En la tabla siguiente se enumeran cuatro tipos de propiedades de selección:

|Clase|DESCRIPCIÓN|
|-----------|-----------------|
|[CMFCPropertyGridProperty (clase)](../../mfc/reference/cmfcpropertygridproperty-class.md)|Propiedad de uso general que se usa para especificar el valor de cadenas, valores booleanos, fechas, etc.|

##  <a name="hittest"></a>  CMFCPropertyGridCtrl::HitTest

Recupera un puntero al objeto de propiedad que corresponde a un elemento de control de cuadrícula de propiedades si un punto especificado está en el elemento. Este método también indica el área en el control de cuadrícula de propiedades que contiene el punto.

```

```

### <a name="parameters"></a>Parámetros

*pt*<br/>
de Punto, en coordenadas de cliente.

*pnArea*<br/>
[in, out] Puntero a una `ClickArea` variable. Cuando este método finaliza, la variable indica el *área de propiedades* que contiene el punto especificado. Para obtener más información sobre un área de propiedades, vea la sección Comentarios.

*bPropsOnly*<br/>
de TRUE para probar solo el área de propiedades; FALSE para probar el *área de Descripción* si el punto especificado no está en el área de propiedades. El valor predeterminado es FALSE. Para obtener más información sobre el área de descripción, vea la sección Comentarios.

Este método establece el valor de la variable a la que apunta el parámetro *pnArea* . En la tabla siguiente se enumeran los valores posibles y las áreas correspondientes.

|Valor|Área|
|-----------|----------|
|`ClickArea::ClickExpandBox`|Control de cuadro de expansión de propiedad.|
|`ClickArea::ClickName`|Nombre de la propiedad.|
|`ClickArea::ClickValue`|Valor de propiedad.|
|`CMFCPropertyGridProperty::ClickDescription`|Área de Descripción del control de cuadrícula de propiedades.|
