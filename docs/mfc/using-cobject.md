# <a name="using-cobject"></a>Usar CObject

[CObject](../mfc/reference/cobject-class.md) es la clase base raíz para la mayoría de los biblioteca MFC (MFC). La clase `CObject` contiene muchas características útiles que puede incluir en sus propios objetos de programa, como la compatibilidad con la serialización, la información de clase en tiempo de ejecución y la salida de diagnóstico de objetos. Si deriva la clase de `CObject`, la clase puede aprovechar estas características `CObject`.
