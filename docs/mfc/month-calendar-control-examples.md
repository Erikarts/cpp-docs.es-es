# <a name="month-calendar-control-examples"></a>Ejemplo de control de calendario mensual

El [CMNCTRL1](../overview/visual-cpp-samples.md) aplicación de ejemplo muestra los distintos atributos de la `CMonthCalCtrl` clase. El control, que se encuentra en una pestaña independiente en el ejemplo, muestra la funcionalidad básica y permite al usuario modificar ciertos atributos de forma dinámica.
