# <a name="using-cstatusbarctrl"></a>Usar CStatusBarCtrl

Puede utilizar el control de barra de estado ([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)) para crear una ventana de control que refleje diversos tipos de información de estado sobre la aplicación. La ventana de estado se puede dividir en partes que muestran más de un tipo de información.
