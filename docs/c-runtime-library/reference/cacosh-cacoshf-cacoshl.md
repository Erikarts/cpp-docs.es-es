# <a name="cacosh-cacoshf-cacoshl"></a>cacosh, cacoshf, cacoshl

Recupera el coseno hiperbólico inverso de un número complejo con un corte de bifurcación en valores inferiores a 1 en el eje real. .

## <a name="syntax"></a>Sintaxis

```C
_Dcomplex cacosh(
   _Dcomplex z
);
_Fcomplex cacosh(
   _Fcomplex z
);  // C++ only
_Lcomplex cacosh(
   _Lcomplex z
);  // C++ only
_Fcomplex cacoshf(
   _Fcomplex z
);
_Lcomplex cacoshl(
   _Lcomplex z
);
```

### <a name="parameters"></a>Parámetros

*z*<br/>
Número complejo que representa un ángulo en radianes.

## <a name="return-value"></a>Valor devuelto

El coseno hiperbólico inverso de *z*, en radianes. El resultado es ilimitado y no negativo en el eje real y en el intervalo [-iπ, + iπ] en el eje imaginario.

## <a name="remarks"></a>Comentarios

Dado que C++ permite las sobrecargas, puede llamar a sobrecargas de **cacosh** que toman y devuelven **_Fcomplex** y **_Lcomplex** valores. En un programa C, **cacosh** siempre toma y devuelve un **_Dcomplex** valor.
