# <a name="_ismbbalpha-_ismbbalpha_l"></a>_ismbbalpha, _ismbbalpha_l

Determina si un carácter multibyte especificado es alfa.

## <a name="syntax"></a>Sintaxis

```C
int _ismbbalpha(
   unsigned int c
);
int _ismbbalpha_l(
   unsigned int c
);
```

### <a name="parameters"></a>Parámetros

*C*<br/>
Entero que se va a probar.

*locale*<br/>
Configuración regional que se va a usar.

## <a name="return-value"></a>Valor devuelto

**_ismbbalpha** devuelve un valor distinto de cero si la expresión:

`isalpha(c) || _ismbbkalnum(c)`

es distinto de cero para *c*, o 0 si no lo es. **_ismbbalpha** utiliza la configuración regional actual para cualquier configuración de caracteres dependiente de la configuración regional. **_ismbbalpha_l** es idéntica, excepto que utiliza la configuración regional pasada.

## <a name="remarks"></a>Observaciones

De forma predeterminada, el estado global de esta función se limita a la aplicación. Para cambiar esto, consulte [Estado global en el CRT](../global-state.md).
