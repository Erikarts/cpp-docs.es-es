# <a name="copysign-copysignf-copysignl-_copysign-_copysignf-_copysignl"></a>copysign, copysignf, copysignl, _copysign, _copysignf, _copysignl

Devuelve un valor que tiene la magnitud de un argumento y el signo de otro.

## <a name="syntax"></a>Sintaxis

```C
double copysign(
   double x,
   double y
);
float copysign(
   float x,
   float y
); // C++ only
long double copysign(
   long double x,
   long double y
); // C++ only
float copysignf(
   float x,
   float y
); // C++ only
long double copysignl(
   long double x,
   long double y
); // C++ only
double _copysign(
   double x,
   double y
);
long double _copysignl(
   long double x,
   long double y
);
```

### <a name="parameters"></a>Parámetros

*x*<br/>
Valor de punto flotante que se devuelve como magnitud del resultado.

*y*<br/>
Valor de punto flotante que se devuelve como signo del resultado.

[Rutinas de compatibilidad con el punto flotante](../../c-runtime-library/floating-point-support.md)

## <a name="return-value"></a>Valor devuelto

Las funciones **copysign** devuelven un valor de punto flotante que combina la magnitud de *x* y el signo de *y*. No se devuelve ningún error.

## <a name="remarks"></a>Comentarios

Dado C++ que permite las sobrecargas, puede llamar a las sobrecargas de **copysign** que toman y devuelven valores de tipo **float** o **Long** **Double** . En un programa de C, **copysign** siempre toma y devuelve un **valor Double**.
