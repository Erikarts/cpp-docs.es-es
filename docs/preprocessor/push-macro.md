# <a name="push_macro-pragma"></a>push_macro (Pragma)

Guarda el valor de la macro de *nombre de macro* en la parte superior de la pila para esta macro.

## <a name="syntax"></a>Sintaxis

> **#pragma push_macro ("** _nombre-macro_ **")**

## <a name="remarks"></a>Comentarios

Puede recuperar el valor de *nombre de macro* con `pop_macro`.

Vea [pop_macro](../preprocessor/pop-macro.md) para obtener un ejemplo.
