# <a name="omp-pragma"></a>omp (Pragma)

Toma una o más directivas de OpenMP, junto con cualquier cláusula directiva opcional.

## <a name="syntax"></a>Sintaxis

> **#pragma OMP** *Directiva* de

## <a name="remarks"></a>Comentarios

Para obtener más información, vea [directivas de OpenMP](../parallel/openmp/reference/openmp-directives.md).
