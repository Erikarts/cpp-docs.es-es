# <a name="auto_search-import-attribute"></a>atributo de importación auto_search

**C++Cuestión**

Especifica que, cuando se hace referencia a una biblioteca de `#import` tipos con y a sí mismo hace referencia a otra biblioteca de tipos `#import` , el compilador puede hacer una implícita para la otra biblioteca de tipos.

## <a name="syntax"></a>Sintaxis

> **#import** *biblioteca de tipos* **auto_search**

## <a name="remarks"></a>Comentarios

**Específico C++ de finalización**
