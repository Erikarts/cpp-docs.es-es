# <a name="catldebuginterfacesmodule-class"></a>Clase CAtlDebugInterfacesModule

Esta clase proporciona compatibilidad con las interfaces de depuración.

## <a name="syntax"></a>Sintaxis

```cpp
class CAtlDebugInterfacesModule
```

## <a name="remarks"></a>Observaciones

`CAtlDebugInterfacesModule`proporciona la compatibilidad necesaria para las interfaces de depuración. Se incluye en cualquier proyecto que defina el símbolo [_ATL_DEBUG_QI](debugging-and-error-reporting-macros.md#_atl_debug_qi).

## <a name="requirements"></a>Requisitos

**Encabezado:** ATLBase. h
