# <a name="error-information-classes"></a>Clases de información de error

La clase siguiente indica cómo se administra la información de error:

- [ISupportErrorInfoImpl](../atl/reference/isupporterrorinfoimpl-class.md) Determina si el objeto admite la interfaz [IErrorInfo](/windows/win32/api/oaidl/nn-oaidl-ierrorinfo) . `IErrorInfo`permite que la información de error se propague de nuevo al cliente.
