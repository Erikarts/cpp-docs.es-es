# <a name="atl-registry-component-registrar"></a>Componente de registro de ATL (Registrador)

El registrador ATL proporciona acceso optimizado al registro del sistema a través de una interfaz personalizada. El registrador es de subproceso libre y permite la vinculación estática de código para los clientes De++ .

> [!NOTE]
> El código fuente del registrador de ATL se puede encontrar en atlmfc-include-atliface.h.
