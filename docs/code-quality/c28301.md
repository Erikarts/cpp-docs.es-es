# <a name="c28301"></a>C28301

> ADVERTENCIA C28301: no hay anotaciones para la primera declaración de \<función >.\<NOTE1 > Consulte \<nombre de archivo > (\<de línea >). \<Note2 >

Esta advertencia se indica cuando no se encontraron anotaciones en la primera declaración de una función determinada.
