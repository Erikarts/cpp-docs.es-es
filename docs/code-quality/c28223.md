# <a name="c28223"></a>C28223

> ADVERTENCIA C28223: no se encontró el token o identificador esperado para la anotación, parámetro

Esta advertencia indica que se espera que un parámetro de una anotación sea un identificador y se encontró algún otro símbolo. Normalmente, esto indica una macro de anotación codificada incorrectamente. El uso de C o C++ palabras clave en esta posición producirá este error.
