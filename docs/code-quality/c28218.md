# <a name="c28218"></a>C28218

> ADVERTENCIA C28218: para el parámetro de función, el parámetro de la anotación no coincide con el que se encuentra en el archivo.

Las anotaciones en la línea actual y en la línea del mensaje son incoherentes. Esto no debería ser posible si las macros estándar se utilizan para las anotaciones. no se espera que esta advertencia se vea en el uso normal.
