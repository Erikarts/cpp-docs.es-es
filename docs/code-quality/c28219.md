# <a name="c28219"></a>C28219

> ADVERTENCIA C28219: miembro de la enumeración esperado para la anotación del parámetro en la anotación

Se espera que un parámetro de una anotación sea miembro del tipo de `enum` con nombre y se haya encontrado algún otro símbolo; Use un miembro de ese `enum` tipo. Normalmente, esto indica una macro de anotación codificada incorrectamente.
