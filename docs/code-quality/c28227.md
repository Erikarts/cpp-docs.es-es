# <a name="c28227"></a>C28227

> ADVERTENCIA C28227: la anotación no puede ser un PrimOp (consulte la declaración anterior)

Esta advertencia indica que la anotación con nombre se declara como una anotación ordinaria y que también se declaró como PrimOp. Esto no debería ser posible si las macros estándar se utilizan para las anotaciones. no se espera que esta advertencia se vea en el uso típico.
