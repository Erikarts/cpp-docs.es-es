# <a name="c26800"></a>C26800

> ADVERTENCIA C26800: uso de un objeto que se ha despasado de Object: \<bloqueo >.

Warning C26800 se desencadena cuando se usa la variable después de haberse pasado de. Una variable se considera movida desde después de pasarla a una función como referencia rvalue. Hay algunas excepciones legítimas para los usos como la asignación, la destrucción y algunas funciones de restablecimiento de estado como STD:: Vector:: CLEAR.

## <a name="example"></a>Ejemplo

El código siguiente generará C26800.

```cpp

#include <utility>

struct X {
    X();
    X(const X&);
    X(X&&);
    X &operator=(X&);
    X &operator=(X&&);
    ~X();
};

template<typename T>
void use_cref(const T&);

void test() {
    X x1;
    X x2 = std::move(x1);
    use_cref(x1);                // @expected(26800)
}
```

## <a name="example"></a>Ejemplo

El código siguiente no generará C26800.

```cpp

#include <utility>

struct MoveOnly {
    MoveOnly();
    MoveOnly(MoveOnly&) = delete;
    MoveOnly(MoveOnly&&);
    MoveOnly &operator=(MoveOnly&) = delete;
    MoveOnly &operator=(MoveOnly&&);
    ~MoveOnly();
};

template<typename T>
void use(T);

void test() {
    MoveOnly x;
    use(std::move(x)); // no 26800
}
```
