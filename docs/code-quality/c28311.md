# <a name="c28311"></a>C28311

> ADVERTENCIA C28311: el < annotation_name > anotación en \<función > \<parámetro > es una versión de obsoleta de SAL.

La anotación es una versión anterior y debe actualizarse a la [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)equivalente. Esta advertencia no se genera si se ha emitido una advertencia de anotación incoherente anterior y se notifica una sola vez por declaración. Inspeccione el resto de la declaración para ver si hay más lenguaje SAL obsoleto.
