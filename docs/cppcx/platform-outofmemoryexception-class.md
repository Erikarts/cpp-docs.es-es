# <a name="platformoutofmemoryexception-class"></a>Platform::OutOfMemoryException (Clase)

Se produce cuando la memoria es insuficiente para completar la operación.

## <a name="syntax"></a>Sintaxis

```cpp
public ref class OutOfMemoryException : COMException,    IException,    IPrintable,    IEquatable
```

### <a name="remarks"></a>Comentarios

Para obtener más información, consulta la clase [COMException](../cppcx/platform-comexception-class.md) .

### <a name="requirements"></a>Requisitos

**Cliente mínimo admitido:** Windows 8

**Servidor mínimo admitido:** Windows Server 2012

**Espacio de nombres**: Plataforma

**Metadatos:** platform.winmd
