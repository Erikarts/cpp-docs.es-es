Los temas de esta sección proporcionan algunos detalles acerca de cómo utilizar el sistema de compilación para generar una aplicación de la plataforma Universal de Windows o un componente de Windows en tiempo de ejecución.

## <a name="in-this-section"></a>En esta sección

- [Bibliotecas estáticas](../cppcx/static-libraries-c-cx.md)

- [DLL](../cppcx/dlls-c-cx.md)

Nota: Visual C++ no es compatible con las optimizaciones guiadas por perfiles para la plataforma Universal de Windows. Si intentas compilar un proyecto con estas opciones establecidas en el IDE, se producirá un error de compilación. Tampoco se admiten las aplicaciones de consola.
